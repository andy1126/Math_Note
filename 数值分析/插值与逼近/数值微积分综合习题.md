# 数值微积分综合习题

本篇收口"数值微分 → 外推 → 自适应 → FFT"主线，以四道综合题串联 [[数值微分]]、[[Romberg-Richardson外推]]、[[自适应求积]]、[[FFT快速傅里叶变换]] 的核心结论。风格沿用 [[矩阵分解综合题解]]。

## 题目

> [!question] 题一（数值微分 + 最优步长）
> 用中心差商估计 $f(x)=\sqrt x$ 在 $x=2$ 的导数。给出最优步长的量级，并解释为何它比 $f(x)=\sin x$（在 $x=1$）情形**更大**。

> [!question] 题二（Romberg 表构造）
> 对 $I=\displaystyle\int_0^{\pi/2}\sin x\,dx=1$ 构造 $4$ 行 Romberg 表，验证第 $3$ 列（$R_{3,3}$）达 $O(h^8)$ 精度。

> [!question] 题三（自适应 Simpson）
> 对 Runge 型积分 $I=\displaystyle\int_0^1\frac{1}{1+25x^2}\,dx$，容差 $10^{-6}$，估算自适应 Simpson 所需函数求值数与复合 Simpson 的对比。

> [!question] 题四（FFT 多项式乘法）
> 用 FFT 计算 $(1+x+x^2)(1+2x+3x^2)$，逐步列出补零、变换、逐点乘、逆变换的全过程。

## 预备定义与定理

> [!note] 工具速查
> - [[数值微分]]：中心差商 $D_0=\tfrac{f(x+h)-f(x-h)}{2h}$，误差 $-\tfrac{h^2}{6}f'''(\xi)$；最优步长 $h^*\sim\bigl(\tfrac{3\varepsilon}{|f'''|}\bigr)^{1/3}$。
> - [[Romberg-Richardson外推]]：$R_{k,m}=\dfrac{4^m R_{k,m-1}-R_{k-1,m-1}}{4^m-1}$，第 $m$ 列 $O(h^{2m+2})$。
> - [[自适应求积]]：局部误差估计 $|S_2-S|/15$，容差按段长加权。
> - [[FFT快速傅里叶变换]]：卷积定理 $\widehat{x\circledast y}=\hat x\odot\hat y$；线性卷积须补零至 $\ge 2n-1$。

## 解答

### 题一

中心差商总误差 $E(h)\approx\tfrac{|f'''|}{6}h^2+\tfrac{\varepsilon}{h}$，最优步长 $h^*=\bigl(\tfrac{3\varepsilon}{|f'''|}\bigr)^{1/3}$。

对 $f=\sqrt x$：$f'(x)=\tfrac12 x^{-1/2}$，$f'''(x)=\tfrac38 x^{-5/2}$，在 $x=2$ 处 $|f'''(2)|=\tfrac38\cdot 2^{-5/2}\approx 0.0663$。取 $\varepsilon\approx\varepsilon_{\text{mach}}=2.2\times10^{-16}$：
$$h^*\approx\Bigl(\frac{3\cdot 2.2\times10^{-16}}{0.0663}\Bigr)^{1/3}\approx(9.9\times10^{-15})^{1/3}\approx 2.2\times10^{-5}.$$

对 $f=\sin x$：在 $x=1$ 处 $|f'''(1)|=|\cos 1|\approx 0.540$，$h^*\approx\bigl(\tfrac{3\varepsilon}{0.54}\bigr)^{1/3}\approx 6\times10^{-6}$。

**为何 $\sqrt x$ 的最优步长更大**：$h^*\propto|f'''|^{-1/3}$。$\sqrt x$ 在 $x=2$ 远离 $x=0$ 的奇异点，$f'''$ 较小（$0.066$）；而 $\sin$ 的 $f'''$ 量级为 $1$。$|f'''|$ 越小，截断误差越温和，可用更大的 $h$ 在舍入主导前取得平衡。（若改在 $x\to 0^+$ 估 $\sqrt x$ 的导数，$f'''$ 发散，$h^*$ 反而极小且精度恶化。）

### 题二

$R_{k,0}$ 为 $[0,\pi/2]$ 上 $N=2^k$ 段复合梯形，逐列外推：

| $k$ | $R_{k,0}$（梯形） | $R_{k,1}$（Simpson） | $R_{k,2}$（Boole） | $R_{k,3}$ |
|---|---|---|---|---|
| 0 | 0.7853981634 | | | |
| 1 | 0.9480594490 | 1.0022798775 | | |
| 2 | 0.9871158010 | 1.0001345850 | 0.9999915655 | |
| 3 | 0.9967851719 | 1.0000082955 | 0.9999998762 | **1.0000000081** |

对角误差链：$R_{0,0}\ 2.1\times10^{-1}\to R_{1,1}\ 2.3\times10^{-3}\to R_{2,2}\ 8.4\times10^{-6}\to R_{3,3}\ 8.1\times10^{-9}$。$R_{3,3}$ 误差 $<10^{-8}$，每列误差降约 $10^{-2.5}$，与第 $m$ 列 $O(h^{2m+2})$ 一致（第 $3$ 列 $O(h^8)$）。

### 题三

被积函数 $\tfrac{1}{1+25x^2}$ 在 $x=0$ 附近最陡（$f'(0)=0$ 但 $f''(0)=-50$，曲率大），向右迅速平缓。

- **复合 Simpson（均匀）**：误差 $\sim\tfrac{(b-a)h^4}{180}\max|f^{(4)}|$，$\max|f^{(4)}|$ 出现在 $x=0$ 附近且很大，为达 $10^{-6}$ 须全程用细步长，约需 $N\approx 60\!\sim\!80$ 段、$120\!\sim\!160$ 次求值。
- **自适应 Simpson**：在 $[0,0.2]$ 多次二分、$[0.4,1]$ 几乎不细分，求值集中在左端。达同容差约需 **$30\!\sim\!40$ 次**求值。

自适应约为复合 Simpson 的 $1/3\!\sim\!1/4$。真值 $I=\tfrac15\arctan 5\approx 0.27468015$。

### 题四

系数向量 $a=(1,1,1)$、$b=(1,2,3)$，乘积次数 $\le 4$，需 $L\ge 2\cdot3-1=5$，取 $2$ 的幂 $L=8$。

1. **补零**：$a=(1,1,1,0,0,0,0,0)$，$b=(1,2,3,0,0,0,0,0)$。
2. **FFT**（$\omega_8=e^{-i\pi/4}$）：$\hat a=\text{FFT}(a)$，$\hat b=\text{FFT}(b)$。例如 $\hat a_0=3,\ \hat b_0=6$（直流分量 = 系数和），其余分量为复数。
3. **逐点乘**：$\hat c_k=\hat a_k\hat b_k$，$k=0,\dots,7$；其中 $\hat c_0=18$。
4. **IFFT**：$c=\text{IFFT}(\hat c)$，取实部四舍五入：
$$c=(1,\,3,\,6,\,5,\,3,\,0,\,0,\,0).$$

即积 $C(x)=1+3x+6x^2+5x^3+3x^4$。验证：直流分量 $\hat c_0=18=\sum c_k=1+3+6+5+3$，与系数和一致。

## 答案

| 题 | 答案 |
|---|---|
| 一 | $f'(2)=\tfrac{1}{2\sqrt2}\approx 0.353553$；$h^*\approx 2\times10^{-5}$，因 $|f'''(2)|\approx0.066$ 小于 $\sin$ 的 $0.54$，$h^*\propto|f'''|^{-1/3}$ 故更大 |
| 二 | $R_{3,3}=1.0000000081$，误差 $8.1\times10^{-9}<10^{-8}$，第 3 列 $O(h^8)$ |
| 三 | 自适应求值数约为复合 Simpson 的 $1/4$；$I\approx 0.274680$ |
| 四 | 乘积系数 $[1,3,6,5,3]$ |

## 变形与延伸

1. **题一变形**：先对中心差商做一次 [[Romberg-Richardson外推|Richardson 外推]]（$O(h^2)\to O(h^4)$），最优步长如何变？由 $h^*\sim\varepsilon^{1/(p+1)}$，$p:2\to4$ 使 $h^*\sim\varepsilon^{1/5}$ 增大，谷底误差降至 $\sim\varepsilon^{4/5}$。
2. **题二变形**：换被积函数为 $e^{-x^2}$（无初等原函数）。仍光滑，Euler–Maclaurin 偶次展开成立，Romberg 同样达高阶精度——可对比 [[Newton-Cotes 数值积分]] 应用 1 的 $e^{x^2}$。
3. **题三变形**：端点奇异 $\int_0^1\ln x\,dx=-1$。$f^{(4)}$ 在 $x\to0^+$ 发散，自适应 Simpson 在左端无限深陷而失效；应改用变量替换 $x=t^2$ 或 Gauss–Jacobi 求积（见 [[Gauss 求积]]、[[自适应求积]] 警告）。
4. **题四变形**：用 FFT 做大整数乘法——把数字按基分块为系数，乘后处理进位。补零长度与循环卷积边界的处理同题四，须 $L\ge 2n-1$ 防回卷。
5. **风格统一**：与 [[矩阵分解综合题解]] 同属综合题解系列，B 模式骨架一致。
