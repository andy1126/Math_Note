# Newton 差商插值

Newton 差商插值与 [[Lagrange插值误差估计|Lagrange 插值]] 在数学上完全等价（同一插值多项式的两种表示），但在算法上有根本优势：**增量友好**——添加新节点时无需重算。差商本身还有清晰的**导数解释**，是连接 [[Hermite 插值]]（重节点情形）的天然桥梁。

## 预备定义

> [!note] 定义 1（差商，divided difference）
> 给定互异节点 $x_0, x_1, \ldots, x_n$ 与函数值 $f_i = f(x_i)$，递归定义：
> - **0 阶差商**：$f[x_i] = f(x_i)$
> - **1 阶差商**：$\displaystyle f[x_i, x_j] = \frac{f[x_j] - f[x_i]}{x_j - x_i}$
> - **$k$ 阶差商**：$\displaystyle f[x_0, \ldots, x_k] = \frac{f[x_1, \ldots, x_k] - f[x_0, \ldots, x_{k-1}]}{x_k - x_0}$

> [!note] 定义 2（Newton 形式插值多项式）
> 满足 $N_n(x_i) = f_i$（$i = 0, \ldots, n$）的次数 $\leq n$ 多项式可写成
> $$N_n(x) = f[x_0] + f[x_0, x_1](x - x_0) + f[x_0, x_1, x_2](x-x_0)(x-x_1) + \cdots$$
> $$+ f[x_0, \ldots, x_n](x-x_0)(x-x_1)\cdots(x-x_{n-1})$$
> 由插值多项式唯一性，$N_n \equiv L_n$（[[Lagrange插值误差估计|Lagrange 形式]]）。

> [!note] 定义 3（前向、后向差分）
> 均匀网格 $x_i = x_0 + ih$ 上：
> - 前向差分 $\Delta f_i = f_{i+1} - f_i$，$\Delta^k f_i = \Delta^{k-1} f_{i+1} - \Delta^{k-1} f_i$
> - 后向差分 $\nabla f_i = f_i - f_{i-1}$
> 二者与差商关系：$f[x_i, \ldots, x_{i+k}] = \dfrac{\Delta^k f_i}{k!\, h^k}$。

> [!note] 定义 4（差商表）
> 用三角阵列组织计算：
>
> | $x_i$ | $f[x_i]$ | 1 阶 | 2 阶 | 3 阶 |
> |---|---|---|---|---|
> | $x_0$ | $f_0$ | | | |
> |       |       | $f[x_0,x_1]$ | | |
> | $x_1$ | $f_1$ | | $f[x_0,x_1,x_2]$ | |
> |       |       | $f[x_1,x_2]$ | | $f[x_0,x_1,x_2,x_3]$ |
> | $x_2$ | $f_2$ | | $f[x_1,x_2,x_3]$ | |
> |       |       | $f[x_2,x_3]$ | | |
> | $x_3$ | $f_3$ | | | |
>
> 每一格 = 右下两格之差 / 横跨 $x$ 之差。Newton 形式取**最上对角线**为系数。

## 引理

> [!abstract] 引理：差商对称性
> 任意置换 $\sigma$，
> $$f[x_{\sigma(0)}, x_{\sigma(1)}, \ldots, x_{\sigma(k)}] = f[x_0, x_1, \ldots, x_k]$$
> 即差商不依赖节点的顺序。

**证明（草要）**：用插值多项式的唯一性。若展开
$$f[x_0, \ldots, x_k] = \sum_{j=0}^k \frac{f(x_j)}{\prod_{i \neq j}(x_j - x_i)}$$
（这是 Lagrange 系数公式），表达式关于下标对称。$\square$

## 主定理

> [!important] 定理：差商与导数
> 若 $f \in C^k[a, b]$，节点 $x_0, \ldots, x_k \in [a, b]$ 互异，则存在 $\xi \in (\min x_i, \max x_i)$ 使
> $$f[x_0, x_1, \ldots, x_k] = \frac{f^{(k)}(\xi)}{k!}$$

**推论（重节点极限）**：当所有节点合并到 $x_0$，
$$\underbrace{f[x_0, x_0, \ldots, x_0]}_{k+1 \text{ 个}} = \frac{f^{(k)}(x_0)}{k!}$$
这是 Newton 差商推广到 [[Hermite 插值]]（含导数信息）的关键。

## 证明

考虑误差函数
$$R(x) = f(x) - N_k(x)$$
$R$ 在 $x_0, \ldots, x_k$ 共 $k+1$ 处取零。由 Rolle 定理迭代 $k$ 次：$R^{(k)}$ 在某 $\xi$ 处为零，即
$$f^{(k)}(\xi) = N_k^{(k)}(\xi)$$
而 $N_k$ 最高次系数恰为 $f[x_0, \ldots, x_k]$（次数 $k$ 项 $f[x_0, \ldots, x_k] \cdot x^k + \cdots$），所以 $N_k^{(k)}(x) = k!\, f[x_0, \ldots, x_k]$。代入：
$$f^{(k)}(\xi) = k!\, f[x_0, \ldots, x_k] \implies f[x_0, \ldots, x_k] = \frac{f^{(k)}(\xi)}{k!} \quad \square$$

## 总结

> [!info] 三种视角理解 Newton 差商
> 1. **代数**：递归定义，差商表 $O(n^2)$ 计算。
> 2. **解析**：差商 $= f^{(k)}(\xi)/k!$，是导数的离散对应。
> 3. **算法**：Newton 形式天然增量，添加节点不破坏已有结果。
>
> 本质上 Newton 与 Lagrange 是**同一多项式的两种基**：Lagrange 用 $\{\ell_i(x)\}$，Newton 用 $\{1, (x-x_0), (x-x_0)(x-x_1), \ldots\}$。后者是**三角基**（下三角变换矩阵），适合递推。

## 应用

### 应用 1：增量插值

需求：已有 $N_n$ 拟合 $\{x_0, \ldots, x_n\}$，新加节点 $x_{n+1}$。

- **Lagrange 方式**：所有 $\ell_i(x)$ 全部需要重算（每个含 $\prod_{j \neq i}(x - x_j)$，多了一因子）。代价 $O(n^2)$。
- **Newton 方式**：差商表追加一行，复用旧值，新增一列对角，代价 $O(n)$。新多项式：
$$N_{n+1}(x) = N_n(x) + f[x_0, \ldots, x_{n+1}](x - x_0)\cdots(x - x_n)$$

这一性质使 Newton 形式成为**自适应插值、误差监控**的首选——可以一边增加节点一边观察新差商绝对值，若已下降到容差以下即可截断。

### 应用 2：具体计算（验证导数关系）

取节点 $(0, 1), (1, 2), (2, 5), (3, 10)$，对应 $f(x) = x^2 + 1$（已知 $f'' \equiv 2$，$f''' \equiv 0$）。

差商表：

| $x_i$ | $f_i$ | 1 阶 | 2 阶 | 3 阶 |
|---|---|---|---|---|
| 0 | 1 | | | |
|   |   | $(2-1)/1 = 1$ | | |
| 1 | 2 | | $(3-1)/2 = 1$ | |
|   |   | $(5-2)/1 = 3$ | | $(1-1)/3 = 0$ |
| 2 | 5 | | $(5-3)/2 = 1$ | |
|   |   | $(10-5)/1 = 5$ | | |
| 3 | 10 | | | |

验证：2 阶差商恒 $= 1 = f''(\xi)/2! = 2/2$，3 阶 $= 0 = f'''(\xi)/3!$。

Newton 多项式：
$$N_3(x) = 1 + 1 \cdot x + 1 \cdot x(x-1) + 0 = 1 + x + x^2 - x = x^2 + 1$$
精确还原 $f$，符合预期。

### 应用 3：差商作为导数估计

取 $f(x) = \sin x$，节点 $\{0, 0.1, 0.2\}$：
- $f[0, 0.1] = (\sin 0.1 - 0)/0.1 \approx 0.99833$
- $f[0.1, 0.2] = (\sin 0.2 - \sin 0.1)/0.1 \approx 0.97843$
- $f[0, 0.1, 0.2] = (0.97843 - 0.99833)/0.2 \approx -0.0995$

由定理，存在 $\xi \in (0, 0.2)$ 使 $f[0, 0.1, 0.2] = f''(\xi)/2 = -\sin(\xi)/2$。
解 $-\sin(\xi)/2 \approx -0.0995 \implies \sin(\xi) \approx 0.199 \implies \xi \approx 0.1003$，确实落在 $(0, 0.2)$ 内。

差商可作为**离散二阶导数估计**——这是 [[Lagrange插值误差估计|插值误差]] 公式 $f^{(n+1)}(\xi)/(n+1)!$ 在数值实践中的具体化。

## 算法关键点

> [!tip] Newton 差商插值实践要诀
> - **等价但不同算**：与 Lagrange 多项式相同，但增量代价 $O(n)$ vs Lagrange 重算 $O(n^2)$。
> - **差商表填表**复杂度 $O(n^2)$ 存储与计算，与 Lagrange 系数相当。
> - **节点排序自由**（对称性），但为数值稳定性，建议按距离评估点 $x$ 由近到远排序——主导项靠前，截断误差小。
> - **均匀节点退化为差分**：若 $h$ 等距，$f[x_0, \ldots, x_k] = \Delta^k f_0/(k!h^k)$，可用 Newton 前向公式 $N_n(x_0 + sh) = \sum_k \binom{s}{k} \Delta^k f_0$。
> - **重节点合并** $\to$ Hermite：$f[x_0, x_0] = f'(x_0)$，自然把导数信息编入差商表。
> - **病态警告**：当节点很密时，差商分母 $x_k - x_0$ 极小，浮点相消严重；考虑 Chebyshev 节点或转 Bernstein 基缓解。

> [!warning] 常见陷阱
> - 节点重复但**未取极限**直接代入定义会除零；编程实现要显式处理重节点分支。
> - Newton 形式系数的**绝对值**并非误差指标；要用差商递归得到的 $f[x_0, \ldots, x_{n+1}](x-x_0)\cdots(x-x_n)$ 的最大绝对值估计余项。

$\blacksquare$
