## 跨板块复用速查

| 复用 | 引用 |
|---|---|
| A1 ↔ A3 | 高斯消元是 LU 的算法构造 |
| A2 ↔ A3 | PLU 是 LU 在主元交换下的推广 |
| B1 ↔ E1 | 谱半径准则已用诱导范数；B1 系统化 |
| B2 ↔ A3, B1 | 用范数+LU 求解分析扰动 |
| C1 ↔ B1 | 正交矩阵保 2-范数 |
| C3 ↔ A4, C1, C4 | 三种最小二乘解法对比 |
| C4 ↔ D1 | SVD 在对称半正定情形即谱分解 |
| D3 ↔ D2 | Gershgorin 定位特征值，幂法精算 |
| D4 ↔ D1, C1 | QR 算法的极限是 Schur 形 |
| E2-E5 ↔ E1 | 一切迭代法收敛性归约到谱半径 |
| E4 ↔ A4 | CG 与 Cholesky 是 SPD 系统的两种解法 |
