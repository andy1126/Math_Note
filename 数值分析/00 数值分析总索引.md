## 板块 II — 微分方程数值解（8 项）

> 从 ODE 单步法、多步法、刚性，过渡到 PDE 有限差分与有限元入门。

## 板块 III — 插值与逼近（12 项）

> 插值 → 数值微分 → 数值积分 → 外推/自适应加速 → 最佳逼近 + 谱方法，节点选择是贯穿主线。

1. [[Lagrange插值误差估计]] — 余项 $\omega(x) f^{(n+1)}/(n+1)!$
2. [[Newton 差商插值]] — Lagrange 的增量等价形式
3. [[Hermite 插值]] — 带导数插值，$\omega^2$ 余项达 $2n+2$ 阶
4. [[三次样条插值]] — $C^2$ 分段三次，最小弯曲性
5. [[Chebyshev 多项式与最佳一致逼近]] — Chebyshev 节点最优 + 交错定理
6. [[数值微分]] — 差商公式族 + 截断/舍入权衡，最优步长 $\sim\varepsilon^{1/(p+1)}$
7. [[Newton-Cotes 数值积分]] — 等距节点积分，复合 Simpson $O(h^4)$
8. [[Gauss 求积]] — 正交多项式根 + $2n-1$ 阶代数精度
9. [[Romberg-Richardson外推]] — Euler-Maclaurin 偶次展开，逐列升两阶
10. [[自适应求积]] — 自适应 Simpson，$|S_2-S|/15$ 局部误差估计
11. [[FFT快速傅里叶变换]] — DFT = 等距三角插值，Cooley-Tukey $O(N\log N)$

综合题解：[[数值微积分综合习题]]

## 板块 IV — 迭代法（6 项）

> 非线性方程根法：从一维到多维，从精确到拟牛顿。

1. [[不动点迭代收敛定理]] — Banach 不动点 + 压缩因子
2. [[Newton法的二阶收敛性]] — 单变量 Newton + 二次收敛
3. [[二分法]] — IVT 保证收敛 + 线性 $1/2$ 收敛
4. [[弦截法]] — 黄金比例 $\varphi \approx 1.618$ 超线性
5. [[多变量 Newton 法]] — Jacobian 法 + Kantorovich 半局部
6. [[拟牛顿法]] — Broyden 秩 1 + BFGS 秩 2 + L-BFGS

## 跨主题衔接

数值分析与其他主题之间的关键 wikilink 路径：

### → ODE 主题
- [[Gronwall不等式]] 是 [[Euler 法]]、[[线性多步法与 Dahlquist 等价定理]]、[[Runge-Kutta方法的收敛性]] 全部收敛证明的离散化基石
- [[Lyapunov稳定性定理]] 与 [[刚性方程与 A-稳定性]] 对应连续 vs 离散稳定性
- [[Sturm-Liouville定理]] 是 [[Gauss 求积]] 中正交多项式族的源头

### → 数学物理方法
- [[热方程基本解]] 与 [[热方程最大值原理]] 在 [[PDE 有限差分法入门]] 中作为连续参照
- [[Legendre多项式的正交性]] 是 [[Gauss 求积]] Gauss-Legendre 节点的依据
- [[Sturm-Liouville特征函数正交性]] 统一 [[Chebyshev 多项式与最佳一致逼近]]、Legendre、Laguerre、Hermite 多项式族

### → 泛函分析
- [[Lax-Milgram定理]] 保证 [[有限元方法入门]] 弱形式的良定性
- [[Sobolev嵌入定理]] 在 FEM 误差估计中提供正则性
- 内积空间与正交投影是 [[最小二乘问题]]、[[Gauss 求积]]、[[Chebyshev 多项式与最佳一致逼近]] 的共通框架

### → 多元微积分
- [[反函数定理]] 与 [[隐函数定理]] 是 [[多变量 Newton 法]] 的几何动机
- Jacobian / Hessian 概念贯穿 [[多变量 Newton 法]]、[[拟牛顿法]]
