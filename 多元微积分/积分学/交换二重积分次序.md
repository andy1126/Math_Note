# 交换二重积分次序

## 题目

> 计算 $\displaystyle \int_0^1 \int_x^1 e^{y^2}\,dy\,dx$.

---

## 预备定义与定理

> [!note] I 型区域与 II 型区域
> 平面有界区域 $D$ 称为 **I 型区域**（$x$-simple），若 $D = \{\, (x,y) \mid a \le x \le b,\; g_1(x) \le y \le g_2(x) \,\}$；
> 称为 **II 型区域**（$y$-simple），若 $D = \{\, (x,y) \mid c \le y \le d,\; h_1(y) \le x \le h_2(y) \,\}$。
> 详见 [[二重积分与Fubini定理]]。

> [!abstract] 定理：Fubini 定理（矩形与一般区域）
> 设 $f$ 在 $D$ 上可积。若 $D$ 为 I 型区域且内层可积，则
> $$\iint_D f = \int_a^b \int_{g_1(x)}^{g_2(x)} f(x,y)\,dy\,dx.$$
> 若 $D$ 为 II 型区域且内层可积，则
> $$\iint_D f = \int_c^d \int_{h_1(y)}^{h_2(y)} f(x,y)\,dx\,dy.$$
> 当被积函数连续（或分段连续）时，两种次序的累次积分均等于二重积分，且可互换。
> 详见 [[二重积分与Fubini定理]]。

> [!note] $e^{y^2}$ 的原函数
> 函数 $y \mapsto e^{y^2}$ 在 $\mathbb{R}$ 上连续，故 Riemann 可积且具有原函数，但该原函数**不是初等函数**（无法用多项式、指数、三角、对数等有限组合表示）。因此 $\int e^{y^2}\,dy$ 不能写成封闭的初等形式。

> [!important] 换序的适用条件
> 本题中被积函数 $e^{y^2}$ 在积分区域（三角形）上连续，满足二重积分的存在性条件；$f \ge 0$ 也可直接用 Tonelli 定理（非负函数无需验证内层可积性）。因此 Fubini 定理完全适用，换序合法。

---

## 解答

### 第一步：写出积分区域

由累次积分的积分限：
$$0 \le x \le 1, \qquad x \le y \le 1.$$

积分区域为 $xy$-平面上以 $(0,0)$, $(1,1)$, $(0,1)$ 为顶点的三角形：
$$D = \{\, (x,y) \mid 0 \le x \le 1,\; x \le y \le 1 \,\}.$$

其边界为：下边界 $y = x$，上边界 $y = 1$，左边界 $x = 0$。

### 第二步：将区域改写为 II 型描述

从 $D$ 的几何形状反解 $x$ 的范围。固定 $y$，$x$ 从 $0$ 变到斜边 $x = y$：

$$D = \{\, (x,y) \mid 0 \le y \le 1,\; 0 \le x \le y \,\}.$$

> [!info] 图示
> $$\begin{tikzpicture}[scale=3]
> \fill[blue!10] (0,0) -- (1,1) -- (0,1) -- cycle;
> \draw[->] (-0.1,0) -- (1.2,0) node[right] {$x$};
> \draw[->] (0,-0.1) -- (0,1.2) node[above] {$y$};
> \draw[thick] (0,0) -- (1,1) node[midway,below right] {$y=x$};
> \draw[thick] (0,1) -- (1,1);
> \draw[dashed] (0,1) -- (0,0);
> \node at (0.3,0.7) {$D$};
> \end{tikzpicture}$$
> 区域 $D$ 由 $y = x$（下边界）、$y = 1$（上边界）和 $x = 0$（左边界）围成。从“先 $y$ 后 $x$”的视角看：$x$ 从 $0$ 到 $1$，对每个 $x$，$y$ 从 $x$ 到 $1$。从“先 $x$ 后 $y$”的视角看：$y$ 从 $0$ 到 $1$，对每个 $y$，$x$ 从 $0$ 到 $y$。

### 第三步：应用 Fubini 定理换序

由 **Fubini 定理**，被积函数 $e^{y^2}$ 在 $D$ 上连续，二重积分与累次积分相等且次序可换：

$$\int_0^1 \int_x^1 e^{y^2}\,dy\,dx = \iint_D e^{y^2}\,dA = \int_0^1 \int_0^y e^{y^2}\,dx\,dy.$$

### 第四步：计算内层积分（对 $x$）

内层积分中 $e^{y^2}$ 与 $x$ 无关（视为常数），直接提出：

$$\int_0^y e^{y^2}\,dx = e^{y^2} \int_0^y 1\,dx = e^{y^2} \cdot \big[x\big]_{x=0}^{x=y} = y\,e^{y^2}.$$

### 第五步：计算外层积分（对 $y$）

$$\int_0^1 y\,e^{y^2}\,dy.$$

做变量替换 $u = y^2$，则 $du = 2y\,dy$，即 $y\,dy = \dfrac{du}{2}$。积分限：$y = 0 \Rightarrow u = 0$，$y = 1 \Rightarrow u = 1$。

$$\int_0^1 y\,e^{y^2}\,dy = \int_0^1 e^u \cdot \frac{du}{2} = \frac12 \big[e^u\big]_{u=0}^{u=1} = \frac{e - 1}{2}.$$

> [!important] 换序是本解法的核心
> 原始累次积分 $\int_x^1 e^{y^2}\,dy$ 无法由初等函数表示（$e^{y^2}$ 无初等原函数），按原次序积分将卡死。换序后内层对 $x$ 积分仅产生因子 $y$，使外层积分 $\int y e^{y^2}\,dy$ 可通过简单的 $u = y^2$ 代换求出。$\blacksquare$

---

## 答案

$$\boxed{\int_0^1 \int_x^1 e^{y^2}\,dy\,dx = \frac{e - 1}{2}.}$$

## 变形与延伸

> [!info] 变形 1（经典姊妹题：$\sin(x^3)$ 需换序）
> 计算 $\displaystyle \int_0^1 \int_{\sqrt{y}}^1 \sin(x^3)\,dx\,dy$。
>
> **提示**：积分区域为 $\{\, (x,y) \mid 0 \le y \le 1,\; \sqrt{y} \le x \le 1 \,\}$。改写为 I 型描述：$y$ 从 $x^2$ 到 $1$ 的反解给出 $0 \le x \le 1,\; 0 \le y \le x^2$。换序后内层对 $y$ 积分仅产生因子 $x^2$：
> $$\int_0^1 \int_0^{x^2} \sin(x^3)\,dy\,dx = \int_0^1 x^2 \sin(x^3)\,dx.$$
> 代换 $u = x^3$ 即得 $\dfrac{1 - \cos 1}{3}$。本题与主解结构完全平行：被积函数对原内层变量无初等原函数，换序后迎刃而解。

> [!info] 变形 2（三次积分换序）
> 计算 $\displaystyle \int_0^1 \int_x^1 \int_y^1 e^{z^2}\,dz\,dy\,dx$。
>
> **提示**：积分区域为 $\{\, (x,y,z) \mid 0 \le x \le 1,\; x \le y \le 1,\; y \le z \le 1 \,\}$。直接积分的最内层 $e^{z^2}$ 无初等原函数。考虑重排积分次序，把对 $z$ 的积分移至最外层。改写区域为 I 型（先 $x$ 后 $y$ 后 $z$ 的其他排列），例如 $0 \le z \le 1$，$0 \le x \le z$，$x \le y \le z$。换为 $\int_0^1 \int_0^z \int_x^z e^{z^2}\,dy\,dx\,dz$，内两层对 $x,y$ 积分仅贡献 $(x,y)$-截面的面积（含 $z$ 的多项式因子），最外层 $\int e^{z^2} \cdot (\text{多项式})\,dz$ 可通过分部积分处理。

> [!info] 变形 3（Fubini 定理的适用条件：换序失败的反例）
> 考虑 $f(x,y) = \dfrac{x^2 - y^2}{(x^2 + y^2)^2}$ 在 $(0,1] \times (0,1]$ 上。计算两种次序的累次积分：
> $$\int_0^1 \left( \int_0^1 f(x,y)\,dx \right) dy \quad\text{与}\quad \int_0^1 \left( \int_0^1 f(x,y)\,dy \right) dx.$$
>
> **提示**：利用 $\dfrac{x^2 - y^2}{(x^2+y^2)^2} = -\dfrac{\partial}{\partial x}\!\left( \dfrac{x}{x^2+y^2} \right)$。对 $x$ 先积分时内层为 $\left[ -\dfrac{x}{x^2+y^2} \right]_{x=0^+}^{x=1} = \dfrac{1}{1+y^2} - \dfrac{1}{y^2}$（$0^+$ 处需小心），再对 $y$ 积分得 $-\dfrac{\pi}{4}$。对 $y$ 先积分时利用 $\dfrac{x^2-y^2}{(x^2+y^2)^2} = \dfrac{\partial}{\partial y}\!\left( \dfrac{y}{x^2+y^2} \right)$，得 $+\dfrac{\pi}{4}$。两种累次积分结果**不相等**，因此 $f$ 的**二重积分不存在**（$|f|$ 在原点附近不可积，不满足 Fubini 定理的可积性前提）。
>
> 这警示：**不能仅凭“两种累次积分都存在”就断言它们相等或二重积分存在**。Fubini 定理要求被积函数在二维区域上可积（或非负时适用 Tonelli 定理），仅累次积分的存在性不足以保证换序合法。

> [!info] 延伸（连接 Tonelli 定理：非负函数的便利性）
> 主解中 $e^{y^2} \ge 0$，适用 **Tonelli 定理**——对非负可测函数，二重积分与两种累次积分（含 $+\infty$ 情形）自动相等，**无需事先验证可积性**。这种“先换序算出有限值，反推可积”的模式是处理非负被积函数的常用策略。
>
> **对比**：若被积函数变号（如变形 3 中的 $f$），Tonelli 不适用，必须验证绝对可积性（$\iint |f| < \infty$）才能安全使用 Fubini 换序。
