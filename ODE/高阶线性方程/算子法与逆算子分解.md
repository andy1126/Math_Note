# 算子法与逆算子分解

## 预备定义

> [!note] 微分算子 $D$ 与算子多项式
> $D := \frac{d}{dx}$。常系数线性 ODE
> $$a_n y^{(n)} + \cdots + a_0 y = g(x)$$
> 可简写为 $p(D) y = g$，其中
> $$p(D) = a_n D^n + \cdots + a_1 D + a_0.$$
> $p(D)$ 是以 $D$ 为"变元"的多项式——常系数保证 $D$ 与数乘可交换，故 $p(D)$ 的代数与普通多项式同构。详见 [[常系数齐次线性方程与特征方程]]。

> [!note] 逆算子 $1/p(D)$
> 形式上定义 $\frac{1}{p(D)}$ 为 $p(D)$ 的**右逆**：
> $$p(D)\left[\frac{1}{p(D)}\, g(x)\right] = g(x).$$
> 即 $y_p = \frac{1}{p(D)} g$ 是 $p(D) y = g$ 的一个特解。逆算子不唯一——差一个 $p(D) y_h = 0$ 的齐次解总是允许的。

> [!abstract] 引理：算子的部分分式分解
> 若 $p(D) = (D - r_1)(D - r_2)\cdots(D - r_n)$（$r_j$ 不必相异），则形式上
> $$\frac{1}{p(D)} = \sum_{j=1}^{s} \sum_{\ell=1}^{m_j} \frac{A_{j\ell}}{(D - r_j)^\ell},$$
> 其中 $m_j$ 为 $r_j$ 的重数，$A_{j\ell}$ 由部分分式分解确定（与 $1/p(s)$ 的复变部分分式系数相同）。这是算子法的代数基石。

> [!abstract] 引理：指数平移定理（exponential shift theorem）
> $$p(D)\big[e^{\alpha x} f(x)\big] = e^{\alpha x}\, p(D + \alpha)\, f(x).$$
> 等价地，
> $$\frac{1}{p(D)}\big[e^{\alpha x} f(x)\big] = e^{\alpha x}\, \frac{1}{p(D + \alpha)}\, f(x).$$
> **证明**：由 Leibniz 公式，$(D - r)[e^{\alpha x} f] = e^{\alpha x} [D + \alpha - r] f$。反复应用即得。

> [!abstract] 引理：基本逆算子公式
> $$\frac{1}{D - r}\, g(x) = e^{rx} \int e^{-rx} g(x)\, dx,$$
> $$\frac{1}{(D - r)^k}\, g(x) = e^{rx} \int\!\!\!\int \!\!\cdots\!\! \int e^{-rx} g(x)\, (dx)^k,$$
> $$\frac{1}{D^k}\, g(x) = \int\!\!\!\int \!\!\cdots\!\! \int g(x)\, (dx)^k \quad (\text{对 } r = 0 \text{ 的特例}).$$

---

## 定理

**(i)（逆算子展开与特解的构造）** 对 $p(D) y = g$，特解可写作
$$y_p = \frac{1}{p(D)}\, g = \sum_{j,\ell} \frac{A_{j\ell}}{(D - r_j)^\ell}\, g,$$
其中系数 $A_{j\ell}$ 来自 $1/p(s)$ 的部分分式分解。每个分项用基本积分公式逐次积分即可。

**(ii)（指数平移定理的应用）** 若 $g(x) = e^{\alpha x} h(x)$，则
$$y_p = e^{\alpha x}\, \frac{1}{p(D + \alpha)}\, h(x),$$
将求解归约为"平移后的逆算子"作用在"除去指数因子"后的函数上——这对右端含 $e^{\alpha x}$ 的情形极其高效。

**(iii)（对三角函数的快速公式）** 对 $g(x) = \cos\beta x$ 或 $\sin\beta x$：
$$\frac{1}{p(D)}\,\cos\beta x = \operatorname{Re}\!\left[\frac{1}{p(i\beta)}\, e^{i\beta x}\right] \quad (p(i\beta) \neq 0),$$
$$\frac{1}{p(D)}\,\sin\beta x = \operatorname{Im}\!\left[\frac{1}{p(i\beta)}\, e^{i\beta x}\right] \quad (p(i\beta) \neq 0).$$
若 $p(i\beta) = 0$（共振），则与指数平移定理结合，引入 $x$ 的因子。

**(iv)（对多项式的快捷公式）** 若 $g(x) = P_m(x)$ 为 $m$ 次多项式且 $p(0) \neq 0$，则将 $1/p(D)$ 形式展开为 $D$ 的 Taylor 级数：
$$\frac{1}{p(D)} = \frac{1}{a_0} \cdot \frac{1}{1 + \frac{a_1}{a_0} D + \cdots} = \frac{1}{a_0} \big(1 + c_1 D + c_2 D^2 + \cdots\big),$$
由于 $D^{m+1} P_m \equiv 0$，只需截断至 $D^m$ 项即可得到多项式特解。

## 证明

### 第(i)部分：部分分式 + 逐项积分

由部分分式分解（复变中的恒等式）：
$$\frac{1}{p(s)} = \sum_{j=1}^{s} \sum_{\ell=1}^{m_j} \frac{A_{j\ell}}{(s - r_j)^\ell}.$$

将 $s$ 替换为 $D$，等式在算子乘法意义下成立（常系数保证 $D$ 与常数交换，算子多项式环同构于普通多项式环）。因此
$$\frac{1}{p(D)}\, g = \sum_{j,\ell} A_{j\ell}\, \frac{1}{(D - r_j)^\ell}\, g.$$

每项 $\frac{1}{D - r_j} g = e^{r_j x} \int e^{-r_j x} g\, dx$（验证：$(D - r_j)[e^{r_j x} \int e^{-r_j x} g] = r_j(\cdots) + e^{r_j x} e^{-r_j x} g - r_j(\cdots) = g$）。重复应用得高阶情形。$\square$

### 第(ii)部分：指数平移

对 $g = e^{\alpha x} h$，由引理：
$$\frac{1}{p(D)}\, e^{\alpha x} h = e^{\alpha x}\, \frac{1}{p(D + \alpha)}\, h.$$

$p(D + \alpha)$ 是 $D$ 的常系数多项式，其系数由 $p$ 的 Taylor 展开 $p(s + \alpha) = \sum \frac{p^{(k)}(\alpha)}{k!} s^k$ 决定。$\square$

### 第(iii)部分：三角函数

将 $\cos\beta x = \operatorname{Re}[e^{i\beta x}]$，由 (ii) 与线性性：
$$\frac{1}{p(D)}\, e^{i\beta x} = e^{i\beta x}\, \frac{1}{p(D + i\beta)}\, 1 = \frac{1}{p(i\beta)}\, e^{i\beta x},$$
因为 $p(D + i\beta) \cdot 1 = p(i\beta) \cdot 1$（$D$ 作用在常数上为 $0$）。

若 $p(i\beta) = 0$（$i\beta$ 是 $m$ 重特征根），则 $p(D) = (D - i\beta)^m q(D)$ 且 $q(i\beta) \neq 0$。由 (i) 的部分分式，含 $\frac{1}{(D - i\beta)^m}$ 的项给出：
$$\frac{1}{(D - i\beta)^m}\, e^{i\beta x} = e^{i\beta x}\, \frac{1}{D^m}\, 1 = e^{i\beta x}\, \frac{x^m}{m!}.$$

取实部/虚部即得共振时的 $x^k \cos$ / $x^k \sin$ 项——与 [[待定系数法]] 的共振规则完全一致。$\square$

### 第(iv)部分：多项式

将 $1/p(D)$ 在 $D = 0$ 附近形式展开（$1/p(s)$ 的 Maclaurin 级数），仅对多项式输入有意义：
$$\frac{1}{p(0) + p'(0) D + \cdots} = \frac{1}{p(0)} \sum_{k=0}^{\infty} b_k D^k.$$

因 $D^{m+1} (x^m) = 0$，级数截断为有限和，给出完全确定的多项式特解。$\blacksquare$

## 例：$\dfrac{1}{D^2 + 1} \sin x$

$p(D) = D^2 + 1$，$p(i) = i^2 + 1 = -1 + 1 = 0$（共振）。$p(s) = (s - i)(s + i)$，部分分式：
$$\frac{1}{s^2 + 1} = \frac{1}{2i}\left(\frac{1}{s - i} - \frac{1}{s + i}\right).$$

$$\frac{1}{D^2 + 1} \sin x = \operatorname{Im}\!\left[\frac{1}{D^2 + 1} e^{ix}\right] = \operatorname{Im}\!\left[\frac{1}{2i}\left(\frac{1}{D - i} - \frac{1}{D + i}\right) e^{ix}\right].$$

$\frac{1}{D - i} e^{ix} = e^{ix} \frac{1}{D} 1 = e^{ix} x$（指数平移 + $\int 1 = x$）。第二项 $\frac{1}{D + i} e^{ix} = e^{ix} \frac{1}{D + 2i} 1 = \frac{e^{ix}}{2i}$（$D$ 作用在常数上为 $0$）。两项组合化简取虚部即得 $y_p = -\frac{x}{2} \cos x$——经典共振解。$\blacksquare$

## 一般解法（算法流程）

> [!tip] 用算子法求 $p(D) y = g$ 的特解
> 1. 写出算子形式，将 $1/p(D)$ 部分分式分解；
> 2. 对每项 $\frac{A}{(D - r)^k}\, g$，用指数平移定理 + 逐次积分；
> 3. 若 $g$ 为多项式，用 $1/p(D)$ 的 Taylor 展开（截断至 $\deg g$）；
> 4. 若 $g = e^{\alpha x} \{\cos, \sin\}\beta x$（无共振），直接用 $\frac{1}{p(\alpha + i\beta)}$ 公式；
> 5. 合并即得 $y_p$。加上 $y_h$（齐次通解）得完整通解。

## 证明思路总结

> [!tip] 关键思想
> 算子法将 [[待定系数法]] 的"猜形式"提升为系统的**代数运算**：
> 1. **部分分式分解**将逆算子 $1/p(D)$ 拆成 $\sum \frac{A}{(D - r)^k}$，每个一阶因子 $1/(D - r)$ = 积分因子法 $\times e^{rx}$（一阶线性方程的核心工具）；
> 2. **指数平移定理** $e^{\alpha x} \frac{1}{p(D + \alpha)} h$ 把"乘以 $e^{\alpha x}$"从"麻烦"变成"助力"——先移走指数，在简化的算子上操作，再移回来；
> 3. **对多项式截断**利用 $D^{m+1} P_m = 0$，将无穷级数变成有限项——纯代数。
>
> 算子法 = 待定系数法 + Laplace 变换法 + 部分分式的融合，是处理**一切**常系数非齐次线性方程的统一代数框架。
