# 降阶法例题

## 题目

> 已知 $y_1 = x$ 是 $x^2 y'' - x(x+2) y' + (x+2) y = 0$ 的一个解，求通解。

---

## 预备定义与定理

> [!note] 标准形（standard form）
> 二阶齐次线性方程除以最高阶系数后写为
> $$y'' + p(x) y' + q(x) y = 0.$$
> 本题需先把 $x^2 y'' - x(x+2)y' + (x+2)y = 0$ 化为此形式（在 $x \neq 0$ 上）。

> [!abstract] 定理：降阶法 / d'Alembert 公式（reduction of order）
> 设 $y_1$ 是 $y'' + p y' + q y = 0$ 的一个非零解。则第二个线性独立解可取
> $$y_2 = y_1 \int \frac{e^{-\int p\,dx}}{y_1^2}\,dx.$$
> 此式中 $e^{-\int p\,dx}$ 恰为 Wronskian（差一常数因子，见 Abel 公式）。详见 [[降阶法]] 与 [[Wronskian与Abel公式]]。

> [!abstract] 引理：Abel 公式给出 $y_2$ 公式的来源
> 由 [[Wronskian与Abel公式]]，$W = y_1 y_2' - y_1' y_2 = C e^{-\int p\,dx}$。把它视为关于 $y_2$ 的一阶线性 ODE，除以 $y_1^2$ 得 $\left(\dfrac{y_2}{y_1}\right)' = \dfrac{W}{y_1^2}$，积分即得上面的 $y_2$ 公式。

> [!abstract] 引理：基本积分
> $$\int \frac{x+2}{x}\,dx = \int\left(1 + \frac{2}{x}\right)dx = x + 2\ln x + C,\qquad \int e^x\,dx = e^x + C.$$

---

## 解答

### 第一步：化为标准形，读出 $p(x)$

在 $x \neq 0$ 上，方程两边除以 $x^2$：
$$y'' - \frac{x(x+2)}{x^2}\,y' + \frac{x+2}{x^2}\,y = 0,$$
即
$$y'' - \frac{x+2}{x}\,y' + \frac{x+2}{x^2}\,y = 0.$$

对照标准形，
$$p(x) = -\frac{x+2}{x},\qquad q(x) = \frac{x+2}{x^2}.$$

### 第二步：计算 $e^{-\int p\,dx}$

先算指数中的积分：
$$-\int p\,dx = -\int\left(-\frac{x+2}{x}\right)dx = \int \frac{x+2}{x}\,dx = x + 2\ln x,$$
（由**基本积分**引理，取常数为 $0$，并设 $x > 0$ 使 $\ln x$ 有定义）。

于是
$$e^{-\int p\,dx} = e^{\,x + 2\ln x} = e^x\cdot e^{2\ln x} = e^x\cdot x^2 = x^2 e^x.$$

> [!info] 这正是 Wronskian（差常数）
> 由 Abel 公式，本题 $W = C e^{-\int p\,dx} = C x^2 e^x$。$y_2$ 公式中的被积函数 $W/y_1^2$ 就来源于此。

### 第三步：代入降阶公式

$y_1 = x$，故 $y_1^2 = x^2$。被积函数
$$\frac{e^{-\int p\,dx}}{y_1^2} = \frac{x^2 e^x}{x^2} = e^x.$$

于是
$$y_2 = y_1\int \frac{e^{-\int p\,dx}}{y_1^2}\,dx = x\int e^x\,dx = x\, e^x.$$
（再次取积分常数为 $0$，因为加常数 $C$ 只会贡献 $C x$，与 $y_1$ 线性相关，被吸收进通解。）

### 第四步：验证 $y_2 = x e^x$

$$y_2 = x e^x,\qquad y_2' = e^x + x e^x = (1 + x)e^x,\qquad y_2'' = e^x + (1+x)e^x = (2 + x)e^x.$$
代入原方程 $x^2 y'' - x(x+2)y' + (x+2)y$：
$$x^2(2+x)e^x - x(x+2)(1+x)e^x + (x+2)\,x e^x.$$
提取公因子 $x(x+2)e^x$：
$$= x(x+2)e^x\big[\,x - (1+x) + 1\,\big] = x(x+2)e^x\cdot 0 = 0. \;\checkmark \quad \square$$

### 第五步：线性独立性与通解

$y_1 = x$ 与 $y_2 = x e^x$ 之比 $y_2/y_1 = e^x$ 非常数，故二者线性独立，构成基本解组（亦可由 $W = x^2 e^x \neq 0$ 于 $x > 0$ 直接判定）。通解为
$$y = C_1 x + C_2 x e^x. \quad \blacksquare$$

---

## 答案

$$\boxed{\,y = C_1 x + C_2 x\, e^x\,}$$

---

## 变形与延伸

> [!info] 变形 1（非齐次）
> 已知 $y_1 = x,\ y_2 = x e^x$，解非齐次方程 $x^2 y'' - x(x+2)y' + (x+2)y = x^3$（$x > 0$）。
>
> **提示**：先化为标准形得右端 $f = x$，再用 [[常数变易法]] 与已求基本解组、$W = x^2 e^x$ 构造特解。

> [!info] 变形 2（另一个已知解）
> 已知 $y_1 = x^2$ 是 $x^2 y'' - 3x y' + 4y = 0$ 的解，用降阶法求第二个解。
>
> **提示**：标准形 $p = -3/x$，$e^{-\int p\,dx} = x^3$，被积函数 $x^3/x^4 = 1/x$，积分得 $\ln x$，故 $y_2 = x^2\ln x$，与 [[Euler方程例题]] 结果一致。

> [!info] 变形 3（直接代换降阶）
> 不背公式，用代换 $y = y_1 v = x v$ 直接降阶，导出关于 $w = v'$ 的一阶方程并求解。
>
> **提示**：代入后 $y_1$ 满足方程的项自动抵消，剩下关于 $v'$ 的可分离 / 一阶线性方程；解出 $v' = e^x$ 后积分得 $v = e^x$，故 $y_2 = x e^x$。这给出公式的"手工"来源。

> [!info] 延伸（高阶降阶）
> 对三阶方程，已知一个解 $y_1$，说明代换 $y = y_1\int w\,dx$ 如何把阶数从 $3$ 降到 $2$。
>
> **提示**：一个已知解把 $n$ 阶线性方程降为关于新未知量的 $(n-1)$ 阶线性方程；本质是商空间 $\text{解空间}/\langle y_1\rangle$ 的维数减一。
