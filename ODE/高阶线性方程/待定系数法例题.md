# 待定系数法例题

## 题目

> 解 $y'' - 2y' + y = e^x\sin x$.

---

## 预备定义与定理

> [!note] 非齐次线性方程的解结构
> 对 $L[y] = f(x)$，通解 $y = y_h + y_p$，其中 $y_h$ 为齐次方程 $L[y] = 0$ 的通解，$y_p$ 为任一特解。

> [!abstract] 定理：待定系数法（method of undetermined coefficients）
> 设 $L = D^2 + a_1 D + a_0$（常系数），右端 $f(x) = e^{\alpha x}(p(x)\cos\beta x + q(x)\sin\beta x)$，$p, q$ 为多项式，$\deg \leq d$。设 $\mu = \alpha + i\beta$ 为特征方程 $P(r) = 0$ 的 $m$ 重根（不是根时 $m = 0$）。则存在特解形如
> $$y_p = x^m e^{\alpha x}\big(\tilde p(x)\cos\beta x + \tilde q(x)\sin\beta x\big),$$
> 其中 $\tilde p, \tilde q$ 为待定的 $\deg \leq d$ 多项式。详见 [[待定系数法]]。

> [!abstract] 引理：指数移位公式（exponential shift）
> 对常系数算子 $L = P(D)$ 与任意充分光滑的 $u$，
> $$L[e^{\alpha x} u] = e^{\alpha x}\, P(D + \alpha)[u].$$
> 特别地，对 $L = (D - 1)^2$ 与 $\alpha = 1$：$P(D + 1) = D^2$，故 $L[e^x u] = e^x u''$。

> [!note] 共振（resonance）
> 当强迫项的复频率 $\mu = \alpha + i\beta$ 恰为特征根时发生共振，需乘 $x^m$（$m$ 为重数）以避免试探解落入齐次解空间。

---

## 解答

### 第一步：解齐次方程

齐次方程 $y'' - 2y' + y = 0$ 的特征方程
$$r^2 - 2r + 1 = (r - 1)^2 = 0,$$
得二重根 $r = 1$。故
$$y_h = (C_1 + C_2 x)\, e^x.$$

### 第二步：判断是否共振

右端 $f(x) = e^x \sin x$ 的复频率为
$$\mu = \alpha + i\beta = 1 + i \cdot 1 = 1 + i.$$

特征根只有 $r = 1$（二重）。由于 $1 + i \neq 1$，**强迫频率 $1 + i$ 不是特征根**，重数 $m = 0$。

> [!important] 无共振，故不乘 $x$
> 共振只在 $\mu$ 等于某特征根时发生。这里 $1 + i$ 的虚部非零，而唯一特征根 $1$ 是实数，二者不等，所以**不需要**乘 $x^m$ 因子。试探解直接取
> $$y_p = e^x(A\cos x + B\sin x).$$

### 第三步：用指数移位化简代入

记 $u = A\cos x + B\sin x$，则 $y_p = e^x u$。算子 $L = D^2 - 2D + 1 = (D - 1)^2$。由**指数移位公式**（$\alpha = 1$）：
$$L[e^x u] = e^x (D + 1 - 1)^2[u] = e^x D^2[u] = e^x u''.$$

> [!info] 为何移位如此干净
> 因为 $1$ 是 $(D-1)^2$ 的二重根，移位后 $P(D + 1) = D^2$ 只剩最高阶导数，省去交叉项的繁琐展开。

### 第四步：求 $u$ 使 $u'' = \sin x$

方程 $L[y_p] = e^x \sin x$ 化为
$$e^x u'' = e^x \sin x \;\Longrightarrow\; u'' = \sin x.$$

对 $u = A\cos x + B\sin x$，
$$u'' = -A\cos x - B\sin x = -u.$$
于是 $-A\cos x - B\sin x = \sin x$。比较系数：
$$-A = 0,\qquad -B = 1 \;\Longrightarrow\; A = 0,\ B = -1.$$

故
$$u = -\sin x,\qquad y_p = e^x(-\sin x) = -e^x\sin x.$$

### 第五步：验证

设 $y_p = -e^x\sin x$。则 $u = -\sin x$，$u'' = \sin x$，故 $L[y_p] = e^x u'' = e^x\sin x$，与右端一致。$\square$

### 第六步：写出通解

$$y = y_h + y_p = (C_1 + C_2 x)\, e^x - e^x\sin x. \quad \blacksquare$$

---

## 答案

$$\boxed{\,y = (C_1 + C_2 x)\, e^x - e^x\sin x\,}$$

---

## 变形与延伸

> [!info] 变形 1（制造共振）
> 解 $y'' - 2y' + y = e^x$。
>
> **提示**：此时强迫频率 $\mu = 1$ 恰为二重特征根，$m = 2$，试探解取 $y_p = A x^2 e^x$。由指数移位 $L[Ax^2 e^x] = e^x (Ax^2)'' = 2A e^x$，得 $A = 1/2$，$y_p = \tfrac12 x^2 e^x$。

> [!info] 变形 2（多项式 × 指数）
> 解 $y'' - 2y' + y = x\, e^x$。
>
> **提示**：$\mu = 1$ 二重根，$m = 2$，试探 $y_p = x^2(Ax + B)e^x$。指数移位给 $u'' = x$，积分两次定 $A, B$（$u = \tfrac16 x^3$，故 $y_p = \tfrac16 x^3 e^x$）。

> [!info] 变形 3（频率不同的三角强迫）
> 解 $y'' - 2y' + y = \cos x$（即 $\alpha = 0$，$\beta = 1$）。
>
> **提示**：$\mu = i$ 不是特征根，试探 $y_p = A\cos x + B\sin x$。代入比较系数得 $-2y' $ 项产生交叉，需解 $2\times 2$ 线性方程组。

> [!info] 延伸（与常数变易法对照）
> 用 [[常数变易法]] 重做本题，验证得到相同 $y_p$。
>
> **提示**：基本解组 $y_1 = e^x,\ y_2 = x e^x$，$W = e^{2x}$；代入 Cramer 公式积分。待定系数法在右端为"指数 × 多项式 × 三角"时更快，常数变易法则对任意连续右端都适用。
