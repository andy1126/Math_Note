# 常数变易法例题

## 题目

> 解 $y'' + y = \sec x$.

---

## 预备定义与定理

> [!note] 二阶非齐次线性方程
> $$y'' + p(x) y' + q(x) y = f(x),$$
> 其中 $p, q, f$ 在区间 $I$ 上连续。本题 $p = 0,\ q = 1,\ f = \sec x$，定义域取 $\left(-\tfrac{\pi}{2}, \tfrac{\pi}{2}\right)$（$\sec x$ 在此连续）。

> [!abstract] 定理：常数变易法（variation of parameters）
> 设 $y_1, y_2$ 为齐次方程的基本解组，Wronskian
> $$W = \begin{vmatrix} y_1 & y_2 \\ y_1' & y_2' \end{vmatrix} = y_1 y_2' - y_1' y_2.$$
> 则非齐次方程有特解
> $$y_p = -y_1 \int \frac{y_2\, f}{W}\,dx + y_2 \int \frac{y_1\, f}{W}\,dx.$$
> 详见 [[常数变易法]]。$W$ 处处非零（基本解组）由 [[Wronskian与Abel公式]] 保证。

> [!abstract] 引理：基本积分
> $$\int \tan x\,dx = -\ln|\cos x| + C,\qquad \int 1\,dx = x + C.$$
> 第一式由 $\dfrac{d}{dx}\ln|\cos x| = \dfrac{-\sin x}{\cos x} = -\tan x$ 立得。

---

## 解答

### 第一步：解齐次方程，确定基本解组

齐次方程 $y'' + y = 0$ 的特征方程 $r^2 + 1 = 0$，根 $r = \pm i$，故
$$y_h = C_1\cos x + C_2\sin x.$$
取基本解组
$$y_1 = \cos x,\qquad y_2 = \sin x.$$

### 第二步：计算 Wronskian

$$W = \begin{vmatrix} \cos x & \sin x \\ -\sin x & \cos x \end{vmatrix} = \cos^2 x + \sin^2 x = 1.$$

> [!info] 与 Abel 公式一致
> 这里 $p = 0$，Abel 公式给 $W = W(x_0)e^{-\int p\,dx} = W(x_0)$ 为常数，确实恒等于 $1$。

### 第三步：套用常数变易公式

右端 $f = \sec x = \dfrac{1}{\cos x}$。代入特解公式（$W = 1$）：
$$y_p = -\cos x \int \frac{\sin x \cdot \sec x}{1}\,dx + \sin x \int \frac{\cos x \cdot \sec x}{1}\,dx.$$

### 第四步：计算两个积分

**第一个积分**：$\sin x \sec x = \dfrac{\sin x}{\cos x} = \tan x$，故由**基本积分**引理
$$\int \sin x\sec x\,dx = \int \tan x\,dx = -\ln|\cos x|.$$

**第二个积分**：$\cos x \sec x = \dfrac{\cos x}{\cos x} = 1$，故
$$\int \cos x\sec x\,dx = \int 1\,dx = x.$$

> [!info] 略去积分常数的理由
> 这两个不定积分的任意常数若保留，会给 $y_p$ 添加 $-C_a\cos x + C_b\sin x$ 形式的项，恰被吸收进齐次通解 $y_h$。故求**特解**时取常数为 $0$ 即可。

### 第五步：组装特解

$$y_p = -\cos x\,(-\ln|\cos x|) + \sin x\,(x) = \cos x\,\ln|\cos x| + x\sin x.$$

### 第六步：验证（可选）

记 $y_p = \cos x\ln|\cos x| + x\sin x$。
$$y_p' = -\sin x\ln|\cos x| + \cos x\cdot\frac{-\sin x}{\cos x} + \sin x + x\cos x = -\sin x\ln|\cos x| + x\cos x.$$
$$y_p'' = -\cos x\ln|\cos x| - \sin x\cdot\frac{-\sin x}{\cos x} + \cos x - x\sin x = -\cos x\ln|\cos x| + \frac{\sin^2 x}{\cos x} + \cos x - x\sin x.$$
于是
$$y_p'' + y_p = \frac{\sin^2 x}{\cos x} + \cos x = \frac{\sin^2 x + \cos^2 x}{\cos x} = \frac{1}{\cos x} = \sec x. \;\checkmark \quad \square$$

### 第七步：写出通解

$$y = C_1\cos x + C_2\sin x + \cos x\,\ln|\cos x| + x\sin x. \quad \blacksquare$$

---

## 答案

$$\boxed{\,y = C_1\cos x + C_2\sin x + \cos x\,\ln|\cos x| + x\sin x\,}$$

---

## 变形与延伸

> [!info] 变形 1（换右端为 $\tan x$）
> 解 $y'' + y = \tan x$。
>
> **提示**：同样 $W = 1$，需算 $\int \sin x\tan x\,dx = \int \dfrac{\sin^2 x}{\cos x}dx = \int(\sec x - \cos x)dx$ 与 $\int \cos x\tan x\,dx = \int\sin x\,dx$。前者用 $\int\sec x\,dx = \ln|\sec x + \tan x|$。

> [!info] 变形 2（换右端为 $\csc x$）
> 在 $(0, \pi)$ 上解 $y'' + y = \csc x$。
>
> **提示**：积分变为 $\int\cos x\csc x\,dx = \int\cot x\,dx = \ln|\sin x|$ 与 $\int\sin x\csc x\,dx = \int 1\,dx = x$。结构与本题对称。

> [!info] 变形 3（待定系数为何失效）
> 说明为什么 $y'' + y = \sec x$ 不能用待定系数法。
>
> **提示**：待定系数法只适用于右端为"指数 × 多项式 × 三角"的有限组合；$\sec x$ 的各阶导数不张成有限维空间，故必须用常数变易法。

> [!info] 延伸（一般二阶公式的推导）
> 推导常数变易法公式：设 $y_p = u_1 y_1 + u_2 y_2$ 并附加约束 $u_1' y_1 + u_2' y_2 = 0$，导出 $u_1', u_2'$ 的 Cramer 解。
>
> **提示**：约束 + 代入方程得线性方程组 $\begin{pmatrix} y_1 & y_2 \\ y_1' & y_2'\end{pmatrix}\begin{pmatrix}u_1'\\u_2'\end{pmatrix} = \begin{pmatrix}0\\f\end{pmatrix}$，解出 $u_1' = -y_2 f/W,\ u_2' = y_1 f/W$，积分即得。$W \neq 0$ 是可解的关键，见 [[Wronskian与Abel公式]]。
