# Euler 方程例题

## 题目

> 解 $x^2 y'' - 3xy' + 4y = 0$.

---

## 预备定义与定理

> [!note] Euler 方程（Cauchy–Euler equation）
> 形如
> $$x^n y^{(n)} + b_{n-1} x^{n-1} y^{(n-1)} + \cdots + b_1 x y' + b_0 y = 0$$
> 的变系数方程，第 $k$ 阶导数前恰配 $x^k$。它在 $x > 0$（或 $x < 0$）上有幂函数解。

> [!abstract] 定理：幂函数试探与指标方程
> 对 Euler 方程在 $x > 0$ 上代入 $y = x^m$，利用 $\dfrac{d^k}{dx^k}x^m = m(m-1)\cdots(m-k+1)\,x^{m-k}$，得**指标方程**（indicial equation）$I(m) = 0$（一个关于 $m$ 的 $n$ 次多项式）。根据根的情形给出基本解组：
> - 单实根 $m$：$x^m$；
> - $k$ 重实根 $m$：$x^m, x^m\ln x, \ldots, x^m(\ln x)^{k-1}$；
> - 复根 $\alpha \pm i\beta$：$x^\alpha\cos(\beta\ln x),\ x^\alpha\sin(\beta\ln x)$。
> 详见 [[Euler方程]]。

> [!abstract] 引理：换元 $x = e^t$ 化为常系数
> 令 $x = e^t$（即 $t = \ln x$，$x > 0$）。则 $x\dfrac{d}{dx} = \dfrac{d}{dt}$，且
> $$x^2\frac{d^2}{dx^2} = \frac{d^2}{dt^2} - \frac{d}{dt}.$$
> Euler 方程因此变为常系数齐次方程，其特征方程即指标方程 $I(m) = 0$。这解释了重根为何产生 $\ln x$ 因子（对应常系数情形的 $t = \ln x$ 因子）。

---

## 解答

### 第一步：代入幂函数试探解

设 $y = x^m$（$x > 0$）。则
$$y' = m x^{m-1},\qquad y'' = m(m-1)x^{m-2}.$$

代入 $x^2 y'' - 3xy' + 4y = 0$：
$$x^2\cdot m(m-1)x^{m-2} - 3x\cdot m x^{m-1} + 4 x^m = 0.$$

每一项的 $x$ 次幂都化为 $x^m$：
$$\big[m(m-1) - 3m + 4\big]\, x^m = 0.$$

> [!info] 次数齐配是 Euler 方程的特征
> 因为第 $k$ 阶导数前配 $x^k$，每项代入后都恰好回到 $x^m$，公因子 $x^m$ 可提出——这正是幂函数试探有效的原因。

### 第二步：写出指标方程

由 $x^m \neq 0$，得**指标方程**
$$I(m) = m(m-1) - 3m + 4 = 0.$$

展开：
$$m^2 - m - 3m + 4 = m^2 - 4m + 4 = 0.$$

### 第三步：求根

$$m^2 - 4m + 4 = (m - 2)^2 = 0 \;\Longrightarrow\; m = 2 \ (\text{二重根}).$$

### 第四步：处理重根，给出基本解组

由**幂函数试探与指标方程**定理，$k = 2$ 重实根 $m = 2$ 贡献两个解
$$y_1 = x^2,\qquad y_2 = x^2\ln x.$$

> [!important] $\ln x$ 因子的来源
> 由换元引理，$x = e^t$ 把方程化为常系数方程 $\ddot{Y} - 4\dot{Y} + 4Y = 0$（其中 $Y(t) = y(e^t)$），特征方程 $(r-2)^2 = 0$ 给二重根 $r = 2$，常系数解为 $(C_1 + C_2 t)e^{2t}$。回代 $t = \ln x,\ e^{2t} = x^2$，即得 $(C_1 + C_2\ln x)x^2$。**重根的第二个解中 $t$ 因子翻译回 $x$ 变量就是 $\ln x$。**

### 第五步：验证 $y_2 = x^2\ln x$

$$y_2' = 2x\ln x + x^2\cdot\frac1x = 2x\ln x + x.$$
$$y_2'' = 2\ln x + 2x\cdot\frac1x + 1 = 2\ln x + 3.$$
代入：
$$x^2(2\ln x + 3) - 3x(2x\ln x + x) + 4x^2\ln x$$
$$= 2x^2\ln x + 3x^2 - 6x^2\ln x - 3x^2 + 4x^2\ln x = (2 - 6 + 4)x^2\ln x + (3 - 3)x^2 = 0. \;\checkmark \quad \square$$

### 第六步：写出通解

$$y = C_1 x^2 + C_2 x^2\ln x = (C_1 + C_2\ln x)\, x^2, \quad x > 0. \quad \blacksquare$$

---

## 答案

$$\boxed{\,y = (C_1 + C_2\ln x)\, x^2,\qquad x > 0\,}$$

---

## 变形与延伸

> [!info] 变形 1（不同实根）
> 解 $x^2 y'' - 2xy' - 4y = 0$（$x > 0$）。
>
> **提示**：指标方程 $m(m-1) - 2m - 4 = m^2 - 3m - 4 = (m-4)(m+1) = 0$，两单根 $m = 4, -1$，通解 $C_1 x^4 + C_2 x^{-1}$。

> [!info] 变形 2（复根）
> 解 $x^2 y'' + xy' + y = 0$（$x > 0$）。
>
> **提示**：指标方程 $m(m-1) + m + 1 = m^2 + 1 = 0$，复根 $m = \pm i$，通解 $C_1\cos(\ln x) + C_2\sin(\ln x)$。

> [!info] 变形 3（非齐次 Euler 方程）
> 解 $x^2 y'' - 3xy' + 4y = x^2$（$x > 0$）。
>
> **提示**：齐次解如本题；因 $x^2$ 对应指标根 $m = 2$ 二重共振，特解试 $y_p = A x^2(\ln x)^2$。或换元 $x = e^t$ 后用待定系数法处理常系数非齐次方程。

> [!info] 延伸（负半轴 $x < 0$）
> 给出 $x < 0$ 上的通解形式。
>
> **提示**：以 $|x|$ 替换，解为 $(C_1 + C_2\ln|x|)\,x^2$；一般地把 $x^m$ 写成 $|x|^m$，$\ln x$ 写成 $\ln|x|$，可统一两个半轴（但 $x = 0$ 为奇点，解不必跨越）。
