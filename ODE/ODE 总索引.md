## 阅读路线

按目标选择路线，括号内为关键前置依赖。

### 路线 A：计算优先（"我能解这个方程"）

> 从"会解一个具体的方程"开始，逐步登高。

```
初等解法/                         高阶线性方程/                   级数解法/
可分离变量 → 一阶线性 → 恰当 →   常系数齐次+特征方程 → 待定系数  常点幂级数 → Airy 例题
Bernoulli → 齐次代换              → 常数变易 → Euler → 降阶     → Frobenius → Bessel 例题
                                  → Laplace 变换 → 算子法       → Legendre/Hermite 例题
                                                               → 超几何/Fuchs 统一
                                       ↓
                                  边值问题/
                                  Green 函数 → 本征值展开
```

- **前置依赖**：[[一阶线性方程与积分因子]]（积分因子思想贯穿始终）
- **理论锚点**：每一步遇到存在/唯一性问题时查阅 [[基本理论/]]

### 路线 B：理论基础（"解为什么存在"）

> 以存在唯一性→解的结构→定性行为为主线。

```
基本理论/                         线性系统/                       Sturm 理论/
Picard-Lindelöf → Peano           Wronskian/Abel → Liouville    Sturm 分离 → Sturm 比较
→ Gronwall → 最大延伸定理         → 矩阵指数 → Floquet          → Sturm-Liouville
→ 连续依赖 → 可微依赖
```

- **衔接**：线性系统的解空间维数论证直接依赖 [[Wronskian与Abel公式]]；SL 定理是线性系统自伴理论的"一维函数版"

### 路线 C：定性分析（"解的行为如何，不解方程也能判断"）

```
稳定性理论/                                   平面动力系统/
Lyapunov 稳定性定理 → LaSalle 不变性原理      Poincaré-Bendixson 定理
→ Hartman-Grobman 定理                       → Bendixson-Dulac 定理
→ 二维相图分类 → Lyapunov 函数构造技巧
```

- **前置**：[[矩阵指数与线性ODE]] 和 [[常系数齐次线性方程与特征方程]] 是相图分类的代数基础
- **例题**：[[相图分类例题]]、[[Lyapunov例题]]

## 板块 I — 基本理论（6 篇）

> 解的存在性、唯一性、参数依赖性——所有初等解法和数值方法的逻辑地基。

1. [[Picard-Lindelöf定理]] — Picard 逐次逼近 + Lipschitz 唯一性
2. [[Peano存在定理]] — 仅需连续性的存在性，紧性论证
3. [[Gronwall不等式]] — 积分/微分形式，先验估计的核心工具
4. [[解的最大延伸定理]] — 解的爆破与最大存在区间
5. [[解对初值的连续依赖性]] — 初值扰动下的解连续变化
6. [[解对初值的可微依赖性]] — 变分方程与流对初值的可微性

## 跨模块关键衔接

ODE 内部各模块之间的核心 wikilink 路径：

### 从计算到理论
- [[可分离变量方程]] → [[Picard-Lindelöf定理]]（分离积分的合法性来自存在唯一性）
- [[一阶线性方程与积分因子]] → [[常数变易法]] → [[两点边值问题与Green函数构造]]（积分因子 → 常数变易 → Green 函数，同一思想的逐步推广）
- [[Laplace变换解常系数线性ODE]] → [[算子法与逆算子分解]] → [[待定系数法]]（三种互补视角：积分变换 / 代数系统 / 试探法）
- [[Bernoulli方程例题]] → [[齐次方程、Bernoulli方程与Riccati方程]] → [[正则奇点与Frobenius方法]]（非线性归约 → Riccati → Sturm-Liouville/Frobenius）

### 从构造到谱
- [[Euler方程]] → [[正则奇点与Frobenius方法]]（Euler 的指标方程是 Frobenius 在常系数极限下的特例）
- [[降阶法]] → [[Wronskian与Abel公式]]（降阶公式 = Abel 公式的构造性推论；Frobenius 第二解含 $\ln$ 项 → 降阶法求 $y_2$）
- [[常点附近的幂级数解]] → [[正则奇点与Frobenius方法]] → [[超几何方程与Fuchs型方程]]（常点 → 正则奇点 → 全纯奇点的统一分类）
- [[Legendre方程的幂级数解]] → [[Sturm-Liouville定理]]（本征值截断 ⇔ SL 边值问题的离散谱）
- [[Hermite方程的级数解]] → [[Sturm-Liouville定理]]（权函数 $e^{-x^2}$ 的无界域 SL 问题）

### 从局部到全局
- [[矩阵指数与线性ODE]] → [[常系数齐次线性方程与特征方程]]（友矩阵的特征多项式 = 标量方程的特征方程）
- [[二维线性系统相图分类]] → [[Hartman-Grobman定理]] → [[Poincaré-Bendixson定理]]（线性分类 → 双曲局部线性化 → 平面全局拓扑）
- [[Lyapunov稳定性定理]] → [[LaSalle不变性原理]] → [[Lyapunov函数构造技巧]]（理论 → 半负定的补救 → 构造实践）
- [[Lyapunov例题]] → [[LaSalle不变性原理]]（$\dot{V} = -y^2 \leq 0$，需 LaSalle 排除 $y=0$ 轴的非平凡不变子集）

### 计算 ↔ 定性的桥梁
- [[常数变易法]] → [[Wronskian与Abel公式]]（$W \neq 0$ 是可解性前提）
- [[Green函数例题]] → [[本征值展开求解非齐次SL问题]]（同一边值问题的积分核 vs 谱分解表示）
- [[超几何方程与Fuchs型方程]] → [[正则奇点与Frobenius方法]]（24 个 Kummer 解 = 基本群表示的不同基底选择）

## 与其他主题的衔接

### → 数值分析
- [[Euler 法]] 的局部截断误差分析用 [[Gronwall不等式]] 做全局误差累积
- [[Gauss 求积]] 的正交多项式节点根源在 [[Legendre方程的幂级数解]]
- [[刚性方程与 A-稳定性]] 与 [[Lyapunov稳定性定理]] 对应离散 vs 连续稳定性

### → 数学物理方法
- [[Sturm-Liouville定理]] 是本征函数展开求解偏微分方程（分离变量法）的核心定理，详见 `数学物理方法/特殊函数与Sobolev/`

- [[Legendre方程的幂级数解]] 连接球坐标下的球谐函数 $Y_\ell^m$

### → 泛函分析
- [[Picard-Lindelöf定理]] 本质上是在 $C^0$ 上的 Banach 不动点定理应用
- [[Lax-Milgram定理]] 保证边值问题弱形式的良定性
- 自伴算子的谱定理 → [[Sturm-Liouville定理]]（一维紧自伴算子的谱）

### → 积分变换与信号处理
- [[Laplace变换解常系数线性ODE]] 连接 [[Laplace 变换的卷积定理]]、[[Laplace变换的初值与终值定理]]、[[LTI系统的BIBO稳定性]]
