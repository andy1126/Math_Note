# 解对初值的连续依赖性

## 预备定义

> [!note] 初值问题（IVP）
> 给定连续函数 $f: U \to \mathbb{R}^n$（$U \subseteq \mathbb{R} \times \mathbb{R}^n$ 为开集）和初始条件 $(t_0, \xi) \in U$，**初值问题**是求函数 $y: I \to \mathbb{R}^n$ 使得
> $$y'(t) = f(t, y(t)), \quad y(t_0) = \xi.$$

> [!note] Lipschitz 条件
> 称 $f: U \to \mathbb{R}^n$ 关于第二个变量满足 **Lipschitz 条件**（Lipschitz 常数为 $L$），若对一切 $(t, y_1), (t, y_2) \in U$，
> $$\|f(t, y_1) - f(t, y_2)\| \leq L \|y_1 - y_2\|.$$

> [!abstract] Gronwall 不等式（积分形式）
> 设 $u, v: [t_0, t_0 + a] \to [0, \infty)$ 连续，$C \geq 0$ 为常数。若对一切 $t \in [t_0, t_0 + a]$，
> $$u(t) \leq C + \int_{t_0}^{t} v(s)\, u(s)\, ds, \quad \cdots (G)$$
> 则
> $$u(t) \leq C \exp\!\left(\int_{t_0}^{t} v(s)\, ds\right) \quad \forall\, t \in [t_0, t_0 + a].$$
> 特别地，若 $v \equiv L$ 为常数，则 $u(t) \leq C \, e^{L(t - t_0)}$。

**证明 Gronwall 不等式**：

定义

$$W(t) = \int_{t_0}^{t} v(s)\, u(s)\, ds.$$

则 $W(t_0) = 0$，且

$$W'(t) = v(t)\, u(t) \leq v(t)\left(C + W(t)\right),$$

其中用到了假设 $(G)$。于是

$$W'(t) - v(t)\, W(t) \leq C\, v(t). \quad \cdots (1)$$

### 应用积分因子

令 $\mu(t) = \exp\!\left(-\int_{t_0}^{t} v(s)\, ds\right)$。将 $(1)$ 两边乘以 $\mu(t) > 0$：

$$\frac{d}{dt}\big[\mu(t)\, W(t)\big] = \mu(t)\left(W'(t) - v(t)\, W(t)\right) \leq C\, v(t)\, \mu(t).$$

从 $t_0$ 积分到 $t$（注意 $\mu(t_0) W(t_0) = 0$）：

$$\mu(t)\, W(t) \leq C \int_{t_0}^{t} v(s)\, \mu(s)\, ds = C\left[-\mu(s)\right]_{s=t_0}^{s=t} = C\left(1 - \mu(t)\right).$$

故

$$W(t) \leq C\left(\frac{1}{\mu(t)} - 1\right) = C\left(e^{\int_{t_0}^{t} v(s)\, ds} - 1\right).$$

代回 $(G)$：

$$u(t) \leq C + W(t) \leq C\, \exp\!\left(\int_{t_0}^{t} v(s)\, ds\right). \quad \square$$

## 定理

设 $f: [t_0, t_0 + a] \times \mathbb{R}^n \to \mathbb{R}^n$ 连续，且关于第二个变量满足 Lipschitz 条件（Lipschitz 常数为 $L$）。设 $y$ 和 $z$ 分别是以下两个初值问题在 $[t_0, t_0 + a]$ 上的解：

$$y' = f(t, y), \quad y(t_0) = \xi,$$
$$z' = f(t, z), \quad z(t_0) = \zeta.$$

则对一切 $t \in [t_0, t_0 + a]$，

$$\|y(t) - z(t)\| \leq \|\xi - \zeta\| \, e^{L(t - t_0)}.$$

特别地，$\xi \to \zeta$ 时 $y \to z$ 一致收敛，即**解连续依赖于初值**。

> [!info] 定理的直观含义
> 初值的微小扰动 $\|\xi - \zeta\|$ 在时间 $t$ 处至多被放大 $e^{L(t-t_0)}$ 倍。增长率由 Lipschitz 常数 $L$ 控制：$L$ 越小，解对初值越"稳定"。这一估计在有限时间区间上是良好的，但在 $t \to \infty$ 时可能呈指数增长——这正是混沌现象的数学根源。

---

## 证明

### 第一步：建立积分方程

由 [[Picard-Lindelöf定理]] 中的等价转化，$y$ 和 $z$ 分别满足积分方程

$$y(t) = \xi + \int_{t_0}^{t} f(s, y(s))\, ds, \qquad z(t) = \zeta + \int_{t_0}^{t} f(s, z(s))\, ds.$$

两式相减：

$$y(t) - z(t) = (\xi - \zeta) + \int_{t_0}^{t} \big[f(s, y(s)) - f(s, z(s))\big]\, ds. \quad \square$$

### 第二步：取范数并应用 Lipschitz 条件

对两边取范数，由三角不等式和 Lipschitz 条件：

$$\|y(t) - z(t)\| \leq \|\xi - \zeta\| + \int_{t_0}^{t} \|f(s, y(s)) - f(s, z(s))\|\, ds \leq \|\xi - \zeta\| + L \int_{t_0}^{t} \|y(s) - z(s)\|\, ds. \quad \cdots (\ast)$$

$\square$

### 第三步：应用 Gronwall 不等式

令 $u(t) = \|y(t) - z(t)\|$，$C = \|\xi - \zeta\|$，$v \equiv L$。则 $(\ast)$ 恰为

$$u(t) \leq C + \int_{t_0}^{t} L\, u(s)\, ds.$$

由 Gronwall 不等式，立即得到

$$\|y(t) - z(t)\| = u(t) \leq C\, e^{L(t - t_0)} = \|\xi - \zeta\|\, e^{L(t - t_0)} \quad \forall\, t \in [t_0, t_0 + a].$$

$\square$

### 推论：解连续依赖于初值

对任意 $\varepsilon > 0$，取 $\delta = \varepsilon\, e^{-La}$。则当 $\|\xi - \zeta\| < \delta$ 时，对一切 $t \in [t_0, t_0 + a]$，

$$\|y(t) - z(t)\| \leq \|\xi - \zeta\|\, e^{La} < \delta\, e^{La} = \varepsilon.$$

故解关于初值的映射 $\xi \mapsto y(\cdot\,; \xi)$ 在 $\|\cdot\|_\infty$ 范数下连续。$\blacksquare$

## 证明思路总结

> [!tip] 关键思想
> 本证明的核心是**三步串联**：
> 1. **积分方程相减**：将两个解的差转化为包含 $\|y - z\|$ 的积分不等式；
> 2. **Lipschitz 条件**：将 $f$ 的差控制为 $y - z$ 的差的线性倍数；
> 3. **Gronwall 不等式**：从"$u \leq C + L\!\int u$"的自引用不等式中解放出显式上界 $Ce^{Lt}$。
>
> Gronwall 不等式是 ODE 理论的**万能放大镜**：它将积分型的隐式估计转化为显式的指数型上界，是证明唯一性、连续依赖性和稳定性的标准工具。

> [!warning] 指数增长的不可避免性与改进
> - **指数增长是尖锐的**：线性系统 $y' = Ly$ 的解为 $y(t) = e^{L(t-t_0)} \xi$，恰好取到等号。故 Gronwall 估计在最坏情形下不可改进。
> - **单边 Lipschitz 条件**：若 $f$ 满足更强的单边条件
> $$\langle f(t, y_1) - f(t, y_2),\, y_1 - y_2 \rangle \leq \ell\, \|y_1 - y_2\|^2 \quad (\ell < L),$$
> 则指数增长率可降为 $e^{\ell(t-t_0)}$，当 $\ell < 0$ 时甚至得到**指数衰减**（耗散系统）。
> - **非自治扰动**：若右端也有扰动 $z' = f(t,z) + g(t)$，类似方法给出
> $$\|y(t) - z(t)\| \leq \left(\|\xi - \zeta\| + \int_{t_0}^{t} \|g(s)\|\, ds\right) e^{L(t-t_0)},$$
> 即解同时连续依赖于初值和右端函数。
