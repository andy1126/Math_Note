# 一阶隐式方程的参数法

## 预备定义

> [!note] 一阶隐式方程
> 形如
> $$F(x, y, y') = 0 \quad \cdots (\mathrm{I})$$
> 的 ODE，其中 $y'$ 不能（或不方便）显式解出为 $y' = f(x, y)$。$F: \mathbb{R}^3 \to \mathbb{R}$ 至少 $C^1$。

> [!note] 参数 $p = y'$
> 标准处理手法是引入参数 $p := y'$，将 (I) 看作 $(x, y, p)$ 空间中的一个曲面 $F(x, y, p) = 0$。在该曲面上，$dy$ 与 $dx$ 满足
> $$dy = p\, dx.$$
> 这是连接三个变量的"桥梁方程"——参数法的所有变体都是在曲面 $F = 0$ 上利用 $dy = p dx$ 将问题化为两个变量的一阶显式方程。

> [!abstract] 引理：隐函数定理
> 设 $F(x, y, p) = 0$ 且在某点 $(x_0, y_0, p_0)$ 处 $\partial F/\partial p \neq 0$，则 $p$ 可局部解为 $(x, y)$ 的函数，从而方程局部化为显式 $y' = f(x, y)$。参数法的意义在于处理**隐函数定理失效**或解出的显式**过于复杂**的情形。

---

## 定理

考虑 $F(x, y, y') = 0$。分三种可解的特殊情形。

**(i)（$y$ 可解出：$y = f(x, p)$）** 若方程可写成
$$y = f(x, p), \quad p = y',$$
则对 $x$ 求导得关于 $p, x$ 的一阶显式方程：
$$p = f_x(x, p) + f_p(x, p) \cdot p'.$$
这是关于 $p(x)$ 的一阶 ODE。解出 $p = \varphi(x, C)$ 后代入 $y = f(x, \varphi)$ 得解。

**(ii)（$x$ 可解出：$x = f(y, p)$）** 若方程可写成
$$x = f(y, p), \quad p = y',$$
则利用 $dy = p\, dx$，将 $dx = f_y dy + f_p dp$ 代入得关于 $p, y$ 的一阶显式方程：
$$dy = p\,(f_y dy + f_p dp) \;\Longrightarrow\; (1 - p f_y)\, dy = p f_p\, dp.$$
这是关于 $p(y)$ 的 ODE。解出后代入 $x = f(y, p)$。

**(iii)（参数化曲面 $F = 0$）** 若能将 $x, y, p$ 同时用一个参数 $t$ 表出：
$$x = X(t), \quad y = Y(t), \quad p = P(t),$$
则由 $dy = p\, dx$ 得 $Y'(t) dt = P(t) X'(t) dt$，即
$$\frac{dY}{dX} = P(t) \quad \text{或} \quad \frac{Y'(t)}{X'(t)} = P(t).$$
这是关于 $t$ 的 ODE。积分得 $t$ 与 $x$（或 $y$）的关系，再回代 $X, Y$ 得参数形式的通解。

---

## 证明

三种情形均为**同一思想的变体**：在曲面 $F(x, y, p) = 0$ 上，将 $dy = p\, dx$ 作为连接三个坐标的约束。

### 第(i)种情形：$y = f(x, p)$

将 $y = f(x, p(x))$ 对 $x$ 求导，并利用 $y' = p$：
$$p = \frac{dy}{dx} = f_x(x, p) + f_p(x, p) \cdot \frac{dp}{dx}.$$

这是关于 $p(x)$ 的一阶 ODE。解出 $p = \varphi(x, C)$ 后，$y = f(x, \varphi(x, C))$ 即原方程的解。$\square$

> [!info] 本情形即 Lagrange 方程
> $y = x g(p) + f(p)$ 是 $y = f(x, p)$ 中 $f$ 对 $x$ 线性的特例。此时 $f_x = g(p)$，$f_p = x g'(p) + f'(p)$，导出的 ODE 为 $\frac{dx}{dp} + \frac{g'(p)}{g(p) - p} x = \frac{f'(p)}{p - g(p)}$——关于 $x(p)$ 是一阶线性方程。

### 第(ii)种情形：$x = f(y, p)$

视 $y$ 为自变量，$x = f(y, p(y))$。求导：
$$dx = f_y\, dy + f_p\, dp.$$

将 $dx$ 代入 $dy = p\, dx$：
$$dy = p\,(f_y dy + f_p dp) = p f_y\, dy + p f_p\, dp.$$

整理：
$$(1 - p f_y)\, dy = p f_p\, dp \;\Longrightarrow\; \frac{dy}{dp} = \frac{p f_p}{1 - p f_y}.$$

这是关于 $y(p)$ 的一阶 ODE。解出 $y = \psi(p, C)$ 后，回代 $x = f(\psi(p, C), p)$ 得参数形式的解（参数为 $p$）。$\square$

### 第(iii)种情形：完全参数化

设 $F(x, y, p) = 0$ 被参数化为 $(X(t), Y(t), P(t))$。沿任意曲线 $t \mapsto (X(t), Y(t))$，若 $Y'(t) = P(t) X'(t)$，则该曲线的 $(x, y)$ 投影满足 $y' = p$，从而是原方程的解。此条件即 $\frac{dY/dt}{dX/dt} = P(t)$，或等价地 $dY = P\, dX$——正是 $dy = p\, dx$ 的参数形式。由此得到关于 $t$ 作为某变量的函数的一阶 ODE。$\blacksquare$

---

## 一般解法（算法流程）

求解 $F(x, y, y') = 0$：

1. **尝试显化**：看能否解出 $y' = f(x, y)$ 或 $y = f(x, p)$ 或 $x = f(y, p)$；
2. **情形 (i)**：若能写成 $y = f(x, p)$（如 Lagrange 方程、Clairaut 方程），对 $x$ 求导，解出 $p$；
3. **情形 (ii)**：若能写成 $x = f(y, p)$（$F$ 不显含 $y$ 或对称形式），以 $y$ 为自变量；
4. **情形 (iii)**：对一般的 $F(x, y, p) = 0$，若前两种不适用，寻找参数化并套用 $dy = p dx$；
5. **回代**：将解出的 $p$（或参数 $t$）代入原关系得 $(x, y)$ 或参数形式的通解。

---

## 证明思路总结

> [!tip] 关键思想
> 一阶隐式方程的全部初等技巧归结为一句几何语言：
>
> **"在 $F(x, y, p) = 0$ 曲面上，沿曲线满足 $dy = p\, dx$。"**
>
> - $y = f(x, p)$ → 以 $x$ 为独立变量，$dy = p dx$ 给出关于 $p(x)$ 的 ODE；
> - $x = f(y, p)$ → 以 $y$ 为独立变量，$dy = p dx$ 给出关于 $p(y)$ 的 ODE；
> - 参数化 $X(t), Y(t), P(t)$ → 以 $t$ 为中间变量，$dY = P dX$ 给出一阶 ODE。
>
> 无论哪种情形，参数 $p = y'$ 的引入让"解不出 $y'$"的困难被转移到"多了一个变量但多了一个微分关系"的平衡中。

> [!info] 方法的适用范围
> 参数法原则上适用于一切一阶隐式方程，但**实际能积出闭形式**的限于：
> - Lagrange 方程 $y = x g(y') + f(y')$（含 Clairaut $g(p) = p$）；
> - 不显含 $y$ 的方程 $F(x, y') = 0$（此时可用 $x$ 作为 $p$ 的函数）；
> - 不显含 $x$ 的方程 $F(y, y') = 0$（此时可用 $y$ 作为 $p$ 的函数）。
>
> 对于更复杂的 $F(x, y, y') = 0$，参数法给出积分形式的隐式解——可视为存在性定理（[[Picard-Lindelöf定理]]）的构造性补充。
