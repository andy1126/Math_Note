```
热方程基本解 → 热方程弱极值原理 → 热方程最大值原理
```

## 第一阶段 — 分离变量法（5 篇）

> 掌握有界区域上三大方程的 Fourier 级数解法。统一四步法：分离假设 → 特征值问题 → 特征函数展开 → 初始条件匹配。

1. [[Fourier级数]] — 正交函数系、Dirichlet 核、逐点收敛定理、$L^2$ 收敛、正弦/余弦展开、Gibbs 现象
2. [[一维热方程的分离变量法]] — $u = XT$ → $\lambda_n = (n\pi/L)^2$ → 杆冷却、非齐次边界稳态化
3. [[一维波动方程的分离变量法]] — 驻波解、特征频率、与热方程对比（特征值相同，时间行为不同）
4. [[矩形Laplace方程的分离变量法]] — 两方向特征展开、混合边界条件、角点奇性
5. [[圆盘Laplace方程与Poisson积分]] — 极坐标分离、Poisson 积分重导出、圆环与扇形区域

## 第三阶段 — 三大方程的理论深入（9 篇）

### 热方程（3 篇）

1. [[热方程基本解]] — 热核 $\Phi(x,t) = (4\pi t)^{-n/2} e^{-|x|^2/4t}$、Duhamel 原理、半空间 Green 函数（奇反射）、Tikhonov 唯一性类
2. [[热方程的弱极值原理]] — 抛物极值原理证明、比较原理、先验估计应用
3. [[热方程最大值原理]] — **强**极值原理（最大值可在边界上取得但在内部传播）、抛物 Harnack 不等式陈述、**无限传播速度**

### 波动方程（3 篇）

4. [[达朗贝尔公式]] — 一维通解、依赖区间与决定区域、半直线问题（反射法）、非齐次 Duhamel 原理

6. [[高维波动方程]] — 三维 Kirchhoff 公式（球平均法）、二维 Poisson 公式（降维法）、Huygens 原理（强 vs 弱）

### 调和分析（3 篇）

7. [[调和函数的均值性质]] — 球面/球体均值定理、Liouville 定理、逆命题完整证明、无穷光滑性

## 第四阶段 — Green 函数方法（2 篇）

> 掌握非齐次方程与一般边界的积分表示。基本思想：$G = \Phi - \phi$（基本解减去调和修正项）。

1. [[Laplace方程Green函数]] — 定义 $-\Delta G = \delta(x-\xi)$ + 齐次边界、对称性 $G(x,\xi)=G(\xi,x)$、镜像法（半空间、球 Kelvin 变换、象限）、Poisson 核 $K = \frac{R^2-|\xi|^2}{n\omega_n R|x-\xi|^n}$
2. [[热方程Green函数]] — 半空间奇反射法、半直线偶反射（Neumann）、有界区间特征展开 $G = \frac{2}{L}\sum \sin\frac{n\pi x}{L}\sin\frac{n\pi \xi}{L} e^{-(n\pi/L)^2 t}$

## 跨模块关键衔接

### 从分离变量到理论性质

- [[Fourier级数]] → [[一维热方程的分离变量法]] → [[热方程最大值原理]]（特征解的 $L^\infty$ 衰减由首项 $e^{-\lambda_1 t}$ 控制 → 极值原理是一致的）
- [[一维波动方程的分离变量法]] → [[达朗贝尔公式]]（驻波叠加 = 行波分解：$\sin(n\pi x/L)\cos(n\pi ct/L) = \frac{1}{2}[\sin(n\pi(x+ct)/L) + \sin(n\pi(x-ct)/L)]$）
- [[圆盘Laplace方程与Poisson积分]] → [[调和函数的均值性质]]（Poisson 积分在 $r \to 0$ 时给出均值性质）

### 从特殊函数到 Green 函数

- [[Sturm-Liouville特征函数正交性]] → [[热方程Green函数]] → [[Laplace方程Green函数]]（S-L 特征展开是 Green 函数的谱表示；镜像法是闭式表示）
- [[Legendre多项式的正交性]] → [[球面调和函数]]（Legendre 多项式是 $m=0$ 的球面调和函数特例）
- [[Bessel函数]] → [[热方程Green函数]]（柱坐标下的特征展开需用 Fourier-Bessel 级数）

### 从古典到现代

- [[Sobolev嵌入定理]] → [[Poisson方程的变分结构]]（极小化序列在 $H^1$ 中的紧性由 Rellich-Kondrachov 保证）
- [[调和函数的极值原理]] → [[Poisson方程的变分结构]]（古典解的唯一性用极值原理；弱解的唯一性用 Lax-Milgram → 两种方法相互印证）
- [[Sturm-Liouville特征函数正交性]] → [[Poisson方程的变分结构]]（Rayleigh 商 → Dirichlet 能量极小化，两者的 $\lambda_1$ 刻画是同一个原理的一维/高维形式）

### 理论内部的分形结构

- [[热方程基本解]] ↔ [[Laplace方程Green函数]]（热核 $\Phi_t$ 是热算子的基本解；Laplace 的基本解 $\Phi$ 是热核的时间积分 $\int_0^\infty \Phi_t dt$）
- [[达朗贝尔公式]] ↔ [[高维波动方程]]（一维行波 = 降维法的终态；三维无尾迹、二维有尾迹的 Huygens 原理差异）
- [[热方程最大值原理]] ↔ [[调和函数的极值原理]]（椭圆的强极值原理是抛物强极值原理在稳态的极限形式）

## 与其他主题的衔接

### ← ODE / Sturm-Liouville 理论
- [[Sturm-Liouville定理]]（`ODE/` 中）是分离变量法中特征值问题的理论基础：自伴性 → 正交性 → 完备性
- [[Bessel方程的Frobenius解]]（`ODE/` 中）连接柱坐标下的 Laplace 算子分离 → [[Bessel函数]]
- [[Legendre方程的幂级数解]]（`ODE/` 中）连接球坐标下的球谐函数 $Y_l^m$ → [[球面调和函数]]
- [[两点边值问题与Green函数构造]]（`ODE/` 中）是一维 Green 函数方法的直接源头

### → 泛函分析
- [[Picard-Lindelöf定理]]（不动点定理）→ [[Poisson方程的变分结构]]（Lax-Milgram 定理 = Riesz 表示 + 强制双线性形式）
- [[Lax-Milgram定理]] 保证椭圆边值问题弱形式的良定性，可推广到非对称形式（Nečas/Babuška 定理）
- [[Sobolev嵌入定理]] 提供 $H^1 \hookrightarrow L^2$ 紧嵌入 → 极小化序列的强收敛性

### → 数值分析
- 分离变量法 → 谱方法的理论基础（用特征函数做基底）
- Green 函数 → 边界元方法（BEM）的理论基础
- Poisson 方程变分结构 → 有限元方法（FEM）的理论基础（Galerkin 近似 + Céa 引理）

### → 物理学
- 热方程 → 扩散过程、Brown 运动（Fokker-Planck 方程）
- 波动方程 → 声学、弹性力学、电磁波（Maxwell 方程）
- Laplace 方程 → 静电学、引力势、不可压缩流体
- 球面调和函数 → 量子力学角动量理论、原子轨道 $s, p, d, f$
- Bessel 函数 → 圆柱波导、圆膜振动、衍射理论
