# PDE 的分类

## 预备定义

> [!note] 偏微分方程（PDE）
> 关于未知函数 $u(x_1, \ldots, x_n)$ 及其偏导数的方程称为**偏微分方程**（Partial Differential Equation）。

> [!note] 阶
> PDE 中出现的最高阶偏导数的阶数称为方程的**阶**。

> [!note] 线性 / 半线性 / 拟线性 / 完全非线性
> 设 $k$ 阶 PDE 的一般形式为 $F(x, u, Du, D^2 u, \ldots, D^k u) = 0$。
> 
> - **线性（Linear）**：方程关于 $u$ 及其所有偏导数是线性的，系数仅依赖 $x$。
>   - 例：$u_t = \Delta u$（热方程），$u_{tt} = c^2 u_{xx}$（波动方程），$\Delta u = 0$（Laplace 方程）
> - **半线性（Semilinear）**：最高阶导数项是线性的，但低阶项可含 $u$ 的非线性函数。
>   - 例：$u_t = \Delta u + u^2$（Fisher 方程）
> - **拟线性（Quasilinear）**：关于最高阶导数是线性的，但系数可依赖于 $u$ 及其低阶导数。
>   - 例：$u_t + u u_x = 0$（无粘 Burgers 方程），$(1+u_x^2)u_{yy} = u_{xx}$（极小曲面方程）
> - **完全非线性（Fully nonlinear）**：最高阶导数以非线性方式出现。
>   - 例：$\det(D^2 u) = f$（Monge-Ampère 方程）

## 二阶 PDE 的分类

### 一般形式

考虑两个自变量 $x, y$ 的二阶半线性 PDE：

$$a u_{xx} + 2b u_{xy} + c u_{yy} + (\text{低阶项}) = 0$$

其中 $a, b, c$ 为 $x, y$（或 $u, u_x, u_y$）的函数。

### 判别式

定义**判别式** $\Delta = b^2 - ac$。在点 $(x_0, y_0)$ 处：

| 条件 | 类型 | 典型代表 | 物理含义 |
|------|------|---------|---------|
| $\Delta < 0$ | **椭圆型**（Elliptic） | $\Delta u = 0$（Laplace） | 平衡态、稳态 |
| $\Delta = 0$ | **抛物型**（Parabolic） | $u_t = u_{xx}$（热方程） | 扩散、耗散 |
| $\Delta > 0$ | **双曲型**（Hyperbolic） | $u_{tt} = c^2 u_{xx}$（波动方程） | 波传播、守恒 |

> [!note] $n$ 个变量的推广
> 对一般二阶线性 PDE：
> $$\sum_{i,j=1}^n a_{ij}(x) \frac{\partial^2 u}{\partial x_i \partial x_j} + \text{（低阶项）} = 0$$
> 在固定点 $x_0$，考虑系数矩阵 $A = (a_{ij})$ 的特征值：
> - **椭圆型**：所有特征值同号（$A$ 正定或负定）
> - **抛物型**：有零特征值，其余同号
> - **双曲型**：一个特征值符号与其他相反，其余同号
> - **超双曲型**（Ultrahyperbolic）：至少两个正特征值和两个负特征值

### 分类流程图

```
PDE → 最高阶项的次数？
  ├─ 一阶 → 特征线法（见 [[特征线法及其应用]]）
  └─ 二阶 → 计算判别式 Δ = b² - ac
       ├─ Δ < 0 → 椭圆型 → Laplace/Poisson 方程
       ├─ Δ = 0 → 抛物型 → 热方程/扩散方程
       └─ Δ > 0 → 双曲型 → 波动方程
```

## 标准形化简

通过变量变换，可将二阶常系数 PDE 化为**标准形**。

### 双曲型标准形（$\Delta > 0$）

引入特征坐标后方程化为：
$$u_{\xi\eta} + \text{（低阶项）} = 0$$

或等价地：
$$u_{\xi\xi} - u_{\eta\eta} + \text{（低阶项）} = 0$$

**例**：波动方程 $u_{tt} = c^2 u_{xx}$ → $u_{\xi\eta} = 0$（见 [[达朗贝尔公式]]）。

### 抛物型标准形（$\Delta = 0$）

方程化为：
$$u_{\xi\xi} + \text{（低阶项）} = 0$$

**例**：热方程 $u_t = u_{xx}$（本身已是抛物标准形）。

### 椭圆型标准形（$\Delta < 0$）

方程化为：
$$u_{\xi\xi} + u_{\eta\eta} + \text{（低阶项）} = 0$$

**例**：Laplace 方程 $\Delta u = 0$（本身已是椭圆标准形）。

## 适定性与初边值条件

不同类型的方程需要不同类型的定解条件：

| 类型 | 典型定解条件 | 不适定问题 |
|------|------------|-----------|
| **椭圆型** | 边界条件（Dirichlet / Neumann / Robin） | 纯初值问题 |
| **抛物型** | 初始条件 + 边界条件（初边值问题） | 纯终值问题（反向热方程） |
| **双曲型** | 初始条件 $\times 2$ + 边界条件 | — |

> [!info] Hadamard 适定性
> 一个问题称为**适定**（well-posed），若：
> 1. **存在性**（existence）：解存在
> 2. **唯一性**（uniqueness）：解唯一
> 3. **稳定性**（stability）：解连续依赖于定解条件

## 习题

1. **分类判定**：判定下列方程的类型：
   (a) $u_{xx} + u_{yy} = 0$
   (b) $u_t = \kappa u_{xx}$（$\kappa > 0$）
   (c) $u_{tt} = c^2 u_{xx}$
   (d) $u_{xx} - 2u_{xy} + u_{yy} = 0$（提示：计算 $\Delta$）
   (e) $u_{xx} + 4u_{xy} + u_{yy} = 0$

2. **标准形**：将方程 $u_{xx} + 2u_{xy} - 3u_{yy} = 0$ 化为标准形，判定其类型。

3. **系数依赖**：方程 $y u_{xx} + u_{yy} = 0$ 在什么区域上是椭圆型？抛物型？双曲型？（Tricomi 方程）

4. **适定性判断**：下列问题是否适定？如不适定，说明理由。
   (a) $\Delta u = 0$ 在圆盘上，$u|_{\partial\Omega} = f$
   (b) $u_t = u_{xx}$ 在 $\mathbb{R}$ 上，$u(x,T) = f(x)$（终值问题）
   (c) $u_t = u_{xx}$ 在 $[0,1]$ 上，$u(x,0) = f(x), u(0,t) = u(1,t) = 0$

5. **$n$ 维分类**：判定 $u_{tt} = \Delta u$（$n$ 维波动方程）的类型。

6. **非线性分类**：方程 $u_t + u u_x = \nu u_{xx}$（$\nu > 0$，粘性 Burgers 方程）的最高阶项决定其类型。它是什么类型？

7. **反例思考**：构造一个混合型方程的例子（在不同区域有不同类型）。

> [!hint]- 部分提示
> - 1(d): 判别式 $\Delta = (-1)^2 - 1 \cdot 1 = 0$，抛物型
> - 1(e): $\Delta = 2^2 - 1 \cdot 1 = 3 > 0$，双曲型
> - 2: 特征方向 $\xi = x+y, \eta = 3x-y$，标准形为 $u_{\xi\eta} = 0$（双曲型）
> - 3: 当 $y < 0$ 时双曲型，$y = 0$ 时抛物型，$y > 0$ 时椭圆型
> - 6: 最高阶项为 $\nu u_{xx}$，抛物型（$\nu > 0$）
