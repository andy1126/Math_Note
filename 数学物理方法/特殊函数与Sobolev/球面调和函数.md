# 球面调和函数

## 预备定义

> [!note] 球坐标下的 Laplace 算子
> 在球坐标 $(r,\theta,\varphi)$（$\theta$ 为余纬角，$0 \leq \theta \leq \pi$；$\varphi$ 为方位角，$0 \leq \varphi < 2\pi$）：
> $$\Delta u = \frac{1}{r^2}\frac{\partial}{\partial r}\left(r^2\frac{\partial u}{\partial r}\right) + \frac{1}{r^2\sin\theta}\frac{\partial}{\partial\theta}\left(\sin\theta\frac{\partial u}{\partial\theta}\right) + \frac{1}{r^2\sin^2\theta}\frac{\partial^2 u}{\partial\varphi^2}$$

> [!note] 连带 Legendre 方程
> $$(1-x^2)\frac{d^2 P}{dx^2} - 2x\frac{dP}{dx} + \left[ l(l+1) - \frac{m^2}{1-x^2} \right] P = 0, \quad x = \cos\theta$$
> 其中 $l = 0, 1, 2, \ldots$，$m = 0, \pm 1, \ldots, \pm l$。

> [!note] 连带 Legendre 函数 $P_l^m(x)$
> $$P_l^m(x) = (1-x^2)^{|m|/2} \frac{d^{|m|}}{dx^{|m|}} P_l(x)$$
> 其中 $P_l(x)$ 是 Legendre 多项式。当 $m \geq 0$ 时通常用此定义；当 $m < 0$ 时：
> $$P_l^{-m}(x) = (-1)^m \frac{(l-m)!}{(l+m)!} P_l^m(x)$$

## 球坐标下的完全分离

### 分离假设

设 $u(r,\theta,\varphi) = R(r) \Theta(\theta) \Phi(\varphi)$，代入 $\Delta u = 0$（或 $\Delta u + \lambda u = 0$），得到三个 ODE。

### 方位角 $\Phi(\varphi)$

$$\Phi'' + m^2 \Phi = 0$$

$2\pi$ 周期性要求 $m = 0, \pm 1, \pm 2, \ldots$：
$$\Phi_m(\varphi) = e^{im\varphi}$$

### 极角 $\Theta(\theta)$

令 $x = \cos\theta$，$P(x) = \Theta(\theta)$：
$$\frac{d}{dx}\left[ (1-x^2) \frac{dP}{dx} \right] + \left[ l(l+1) - \frac{m^2}{1-x^2} \right] P = 0$$

这是**连带 Legendre 方程**。要求在 $x = \pm 1$（$\theta = 0, \pi$）处有限 → $l = 0, 1, 2, \ldots$，$|m| \leq l$。解为 $P_l^m(\cos\theta)$。

### 径向 $R(r)$

$R(r)$ 满足 Euler 方程：
$$r^2 R'' + 2r R' - l(l+1) R = 0$$

设 $R(r) = r^\alpha$，得 $\alpha(\alpha+1) - l(l+1) = 0$，故 $\alpha = l$ 或 $\alpha = -(l+1)$。
$$R_l(r) = A_l r^l + B_l r^{-(l+1)}$$

> [!info] 物理选择
> - **内部问题**（$r < a$，含原点）：取 $r^l$（$r^{-(l+1)}$ 在原点发散）
> - **外部问题**（$r > a$，含无穷远）：取 $r^{-(l+1)}$（$r^l$ 在无穷远处发散）

## 球面调和函数 $Y_l^m$

### 定义

将角向部分组合为**球面调和函数** $Y_l^m(\theta,\varphi)$：
$$Y_l^m(\theta,\varphi) = N_l^m P_l^{|m|}(\cos\theta) e^{im\varphi}$$

其中归一化常数（Condon-Shortley 相位约定）：
$$N_l^m = (-1)^m \sqrt{\frac{2l+1}{4\pi} \frac{(l-|m|)!}{(l+|m|)!}}$$

### 正交完备性

在单位球面 $\mathbb{S}^2$ 上：
$$\int_0^{2\pi} \int_0^\pi Y_l^m(\theta,\varphi) \overline{Y_{l'}^{m'}(\theta,\varphi)} \sin\theta \, d\theta \, d\varphi = \delta_{ll'} \delta_{mm'}$$

完备性：任意 $L^2(\mathbb{S}^2)$ 函数可展开为：
$$f(\theta,\varphi) = \sum_{l=0}^\infty \sum_{m=-l}^l a_{lm} Y_l^m(\theta,\varphi), \quad a_{lm} = \int_{\mathbb{S}^2} f(\theta,\varphi) \overline{Y_l^m(\theta,\varphi)} \, d\Omega$$

### 前几个球面调和函数

| $l$ | $m$ | $Y_l^m(\theta,\varphi)$ |
|-----|-----|------------------------|
| $0$ | $0$ | $\frac{1}{\sqrt{4\pi}}$ |
| $1$ | $0$ | $\sqrt{\frac{3}{4\pi}}\cos\theta$ |
| $1$ | $\pm 1$ | $\mp\sqrt{\frac{3}{8\pi}}\sin\theta \, e^{\pm i\varphi}$ |
| $2$ | $0$ | $\sqrt{\frac{5}{16\pi}}(3\cos^2\theta - 1)$ |
| $2$ | $\pm 1$ | $\mp\sqrt{\frac{15}{8\pi}}\sin\theta\cos\theta \, e^{\pm i\varphi}$ |
| $2$ | $\pm 2$ | $\sqrt{\frac{15}{32\pi}}\sin^2\theta \, e^{\pm 2i\varphi}$ |

## 应用：球上 Dirichlet 问题

### 球内 Dirichlet 问题

对 $r < a$，求调和函数使 $u(a,\theta,\varphi) = f(\theta,\varphi)$。

展开 $f$ 在球面上为球谐级数：$f(\theta,\varphi) = \sum_{l,m} a_{lm} Y_l^m(\theta,\varphi)$。

径向部分需在 $r=0$ 有限，故取 $r^l$：
$$u(r,\theta,\varphi) = \sum_{l=0}^\infty \sum_{m=-l}^l a_{lm} \left(\frac{r}{a}\right)^l Y_l^m(\theta,\varphi)$$

这称为**球的 Poisson 积分公式**（级数形式）。

### 球外 Dirichlet 问题

对 $r > a$，解在无穷远处有界，取 $r^{-(l+1)}$：
$$u(r,\theta,\varphi) = \sum_{l=0}^\infty \sum_{m=-l}^l a_{lm} \left(\frac{a}{r}\right)^{l+1} Y_l^m(\theta,\varphi)$$

## 物理应用：多极展开

> [!info] 静电位的多极展开
> 点电荷分布 $\rho(\mathbf{r}')$ 在 $\mathbf{r}$ 处产生的电位：
> $$\Phi(\mathbf{r}) = \frac{1}{4\pi\varepsilon_0} \int \frac{\rho(\mathbf{r}')}{|\mathbf{r} - \mathbf{r}'|} d^3r'$$
> 
> 利用 $\frac{1}{|\mathbf{r}-\mathbf{r}'|} = 4\pi \sum_{l=0}^\infty \sum_{m=-l}^l \frac{1}{2l+1} \frac{r_<^l}{r_>^{l+1}} Y_l^m(\theta,\varphi) \overline{Y_l^m(\theta',\varphi')}$（其中 $r_< = \min(r,r')$，$r_> = \max(r,r')$），电位可展开为：
> $$\Phi(\mathbf{r}) = \frac{1}{\varepsilon_0} \sum_{l,m} \frac{q_{lm}}{2l+1} \frac{Y_l^m(\theta,\varphi)}{r^{l+1}}$$
> 
> $l=0$ 对应**单极子**（总电荷），$l=1$ 对应**偶极子**，$l=2$ 对应**四极子**……

## 习题

1. **前几个 $Y_l^m$ 的验证**：验证 $Y_1^1(\theta,\varphi)$ 和 $Y_2^0(\theta,\varphi)$ 满足正交关系 $\int_{\mathbb{S}^2} Y_1^1 \overline{Y_2^0} d\Omega = 0$。

2. **球内 Dirichlet 问题**：求单位球内调和的函数，边界条件 $u(1,\theta,\varphi) = \cos\theta$。即用球谐展开求出所有 $r$ 处的值。

3. **连带 Legendre 验证**：验证 $P_2^1(x) = -3x\sqrt{1-x^2}$ 满足连带 Legendre 方程（$l=2, m=1$）。

4. **轴对称情形**：若边界条件轴对称（即不依赖 $\varphi$），则展开式中仅 $m=0$ 项非零。证明此时球内 Dirichlet 问题的解简化为 $u(r,\theta) = \sum_{l=0}^\infty A_l r^l P_l(\cos\theta)$。

5. **球外 Dirichlet 问题**：求单位球外部的调和函数，边界条件 $u(1,\theta) = \sin^2\theta$，且在无穷远处 $u \to 0$。提示：$\sin^2\theta = \frac23 P_0(\cos\theta) - \frac23 P_2(\cos\theta)$。

6. **球壳问题**：球壳 $a < r < b$ 内调和，$u(a,\theta) = f(\theta)$，$u(b,\theta) = g(\theta)$（轴对称）。写出解的一般形式，说明径向部分同时包含 $r^l$ 和 $r^{-(l+1)}$。

7. **特征值问题**：单位球面上的 Laplace-Beltrami 算子 $\Delta_{\mathbb{S}^2} Y = -l(l+1)Y$。证明 $\Delta_{\mathbb{S}^2} = \frac{1}{\sin\theta}\frac{\partial}{\partial\theta}(\sin\theta\frac{\partial}{\partial\theta}) + \frac{1}{\sin^2\theta}\frac{\partial^2}{\partial\varphi^2}$ 是自伴算子，特征值为 $-l(l+1)$，重数为 $2l+1$。

> [!hint]- 部分提示
> - 2: $u(r,\theta,\varphi) = r\cos\theta$
> - 3: 从 $P_2(x)=\frac12(3x^2-1)$ 出发，$P_2^1(x) = \sqrt{1-x^2} P_2'(x) = \sqrt{1-x^2} \cdot 3x = 3x\sqrt{1-x^2}$（注：Condon-Shortley 相位含 $(-1)^m$，即 $-3x\sqrt{1-x^2}$）
> - 5: $u(r,\theta) = \frac{2}{3r} P_0(\cos\theta) - \frac{2}{3r^3} P_2(\cos\theta)$
> - 7: 在 $\mathbb{S}^2$ 上做分部积分
