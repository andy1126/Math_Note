# 一维波动方程的分离变量法

## 预备定义

> [!note] 两端固定弦的振动问题
> 考虑长为 $L$ 的均匀弦，两端固定。位移 $u(x,t)$ 满足：
> $$\begin{cases}
> u_{tt} = c^2 u_{xx}, & 0 < x < L, \; t > 0 \\[4pt]
> u(0,t) = u(L,t) = 0, & t > 0 \quad \text{(固定端点)} \\[4pt]
> u(x,0) = f(x), \; u_t(x,0) = g(x), & 0 < x < L
> \end{cases}$$
> 其中 $c = \sqrt{T/\rho}$（$T$：张力，$\rho$：线密度）。

---

## 四步法求解

### 第一步：分离假设

设 $u(x,t) = X(x) T(t)$，代入 $u_{tt} = c^2 u_{xx}$：
$$X(x) T''(t) = c^2 X''(x) T(t)$$

分离变量：
$$\frac{T''(t)}{c^2 T(t)} = \frac{X''(x)}{X(x)} = -\lambda$$

得到两个 ODE：
$$X''(x) + \lambda X(x) = 0 \quad \cdots (1)$$
$$T''(t) + c^2 \lambda T(t) = 0 \quad \cdots (2)$$

### 第二步：特征值问题

由边界条件 $u(0,t) = u(L,t) = 0$ 得 $X(0) = X(L) = 0$。

与热方程情形完全相同：
$$\begin{cases} X''(x) + \lambda X(x) = 0 \\ X(0) = X(L) = 0 \end{cases}$$

解得：
$$\lambda_n = \left(\frac{n\pi}{L}\right)^2, \quad X_n(x) = \sin\frac{n\pi x}{L}, \quad n = 1, 2, 3, \ldots$$

### 第三步：时间部分的振动解

对每个 $\lambda_n$，代入 (2)：$T''(t) + \omega_n^2 T(t) = 0$，其中 $\omega_n = c\sqrt{\lambda_n} = \frac{n\pi c}{L}$。

通解（简谐振动）：
$$T_n(t) = A_n \cos(\omega_n t) + B_n \sin(\omega_n t)$$

> [!note] 特征频率
> $\omega_n = n\pi c/L$ 称为弦的**固有圆频率**（angular eigenfrequency）。
> $f_n = \omega_n/(2\pi) = nc/(2L)$ 是**固有频率**（Hz）。
> $n=1$ 对应**基频**（fundamental），$n \geq 2$ 对应**泛频**（overtones）。

### 第四步：叠加与初始条件匹配

叠加得到通解：
$$u(x,t) = \sum_{n=1}^\infty \sin\frac{n\pi x}{L} \left[ A_n \cos(\omega_n t) + B_n \sin(\omega_n t) \right]$$

代入初始位移 $u(x,0) = f(x)$：
$$f(x) = \sum_{n=1}^\infty A_n \sin\frac{n\pi x}{L} \quad \Rightarrow \quad A_n = \frac{2}{L} \int_0^L f(x) \sin\frac{n\pi x}{L} \, dx$$

对 $t$ 求导得速度：
$$u_t(x,t) = \sum_{n=1}^\infty \sin\frac{n\pi x}{L} \left[ -\omega_n A_n \sin(\omega_n t) + \omega_n B_n \cos(\omega_n t) \right]$$

代入初始速度 $u_t(x,0) = g(x)$：
$$g(x) = \sum_{n=1}^\infty \omega_n B_n \sin\frac{n\pi x}{L} \quad \Rightarrow \quad B_n = \frac{2}{\omega_n L} \int_0^L g(x) \sin\frac{n\pi x}{L} \, dx$$

## 驻波解释

> [!info] 驻波（Standing Waves）
> 解中每个模态：
> $$u_n(x,t) = \sin\frac{n\pi x}{L} \left[ A_n \cos(\omega_n t) + B_n \sin(\omega_n t) \right]$$
> 
> 可以重写为：
> $$u_n(x,t) = C_n \sin\frac{n\pi x}{L} \sin(\omega_n t + \delta_n)$$
> 
> 其中 $C_n = \sqrt{A_n^2 + B_n^2}$，相位 $\delta_n$ 由 $\tan\delta_n = A_n/B_n$ 确定。
> 
> - **波节**（nodes）：$\sin(n\pi x/L) = 0$ 的点 → $x = kL/n$（$k = 0, 1, \ldots, n$）
> - **波腹**（antinodes）：$\sin(n\pi x/L) = \pm 1$ 的点 → $x = (2k+1)L/(2n)$
> - 第 $n$ 模态有 $n+1$ 个节点（含端点）和 $n$ 个腹点

## 与热方程分离变量法的对比

| 特征 | 热方程 $u_t = \kappa u_{xx}$ | 波动方程 $u_{tt} = c^2 u_{xx}$ |
|------|----------------------------|-------------------------------|
| 特征值问题 | 相同（$X'' + \lambda X = 0$ + 边界条件） | 完全相同 |
| 时间方程 | $T' + \kappa\lambda T = 0$（一阶） | $T'' + c^2\lambda T = 0$（二阶） |
| 时间行为 | 指数衰减 $e^{-\kappa\lambda_n t}$ | 等幅振荡 $\sin(\omega_n t + \delta_n)$ |
| 初始条件 | 一个（位移） | 两个（位移 + 速度） |
| 能量 | 耗散（$E \to 0$） | 守恒 |
| 长时间行为 | 趋于零 | 持续振动 |

## 例题

> [!example] 拨弦问题
> 弦初始静止在三角形位置：$f(x) = \begin{cases} \frac{2h}{L}x, & 0 \leq x < L/2 \\[4pt] \frac{2h}{L}(L-x), & L/2 \leq x \leq L \end{cases}$，$g(x) = 0$。
> 
> 由于 $g(x) = 0$，所有 $B_n = 0$。计算 $A_n$（分段积分）：
> $$A_n = \frac{8h}{n^2\pi^2} \sin\frac{n\pi}{2}$$
> 
> 当 $n$ 为偶数时 $A_n = 0$；$n = 1, 5, 9, \ldots$ 时 $A_n > 0$；$n = 3, 7, 11, \ldots$ 时 $A_n < 0$。
> 
> $$u(x,t) = \frac{8h}{\pi^2} \sum_{k=0}^\infty \frac{(-1)^k}{(2k+1)^2} \sin\frac{(2k+1)\pi x}{L} \cos\!\left(\frac{(2k+1)\pi c}{L} t\right)$$
> 
> 注意到拨弦的中点条件（对称），仅奇数模态被激发。

## 习题

1. **基本求解**：求解 $u_{tt} = 4 u_{xx}$，$0 < x < \pi$，$u(0,t) = u(\pi,t) = 0$，$u(x,0) = \sin(3x)$，$u_t(x,0) = 0$。指出固有频率。

2. **纯速度初值**：弦初始静止在平衡位置但受到敲击：$f(x) = 0$，$g(x) = \delta(x - x_0)$（理想化敲击）。写出解的形式（需要分布意义下的 Fourier 系数）。

3. **对比解析解**：对 $f(x) = A\sin(\pi x/L)$，$g(x) = 0$，同时用分离变量法和达朗贝尔公式（需做奇周期延拓）求解，验证结果一致。

4. **能量验证**：利用 Parseval 等式证明弦的总能量 $E = \frac{1}{2}\int_0^L (u_t^2 + c^2 u_x^2)dx = \frac{L}{4}\sum_{n=1}^\infty \omega_n^2 (A_n^2 + B_n^2)$，并验证能量守恒。

5. **共振分析**：若对弦施加频率为 $\omega$ 的周期性外力，什么频率下会发生共振？

6. **非齐次边界**：求解 $u_{tt} = c^2 u_{xx}$，$0 < x < L$，$u(0,t) = 0$，$u(L,t) = A\sin(\Omega t)$（一端振动），$u(x,0) = u_t(x,0) = 0$。提示：令 $u(x,t) = v(x,t) + w(x,t)$，其中 $w$ 满足非齐次边界但满足波动方程。

> [!hint]- 部分提示
> - 1: $u(x,t) = \cos(6t)\sin(3x)$，固有频率 $f_3 = 3c/(2\pi) = 3/\pi$（$c=2$）
> - 2: $B_n = \frac{2}{\omega_n L} \sin(n\pi x_0/L)$
> - 3: 奇周期延拓 $f_{\text{odd}}$ 后使用达朗贝尔公式，再利用三角恒等式展开即得级数形式
> - 5: 共振发生于 $\omega = \omega_n = n\pi c/L$（即外力频率等于任意固有频率）
