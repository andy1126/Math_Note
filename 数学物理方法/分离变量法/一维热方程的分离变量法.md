# 一维热方程的分离变量法

## 预备定义

> [!note] 一维热方程的初边值问题
> 考虑长为 $L$ 的均匀细杆，两端保持零温度。温度分布 $u(x,t)$ 满足：
> $$\begin{cases}
> u_t = \kappa u_{xx}, & 0 < x < L, \; t > 0 \\[4pt]
> u(0,t) = u(L,t) = 0, & t > 0 \quad \text{(齐次 Dirichlet 边界)} \\[4pt]
> u(x,0) = f(x), & 0 < x < L \quad \text{(初始温度)}
> \end{cases}$$
> 其中 $\kappa > 0$ 为热扩散系数。

---

## 四步法求解

### 第一步：分离假设

假设解可以写为仅含 $x$ 的函数和仅含 $t$ 的函数的乘积：
$$u(x,t) = X(x) T(t)$$

代入热方程 $u_t = \kappa u_{xx}$：
$$X(x) T'(t) = \kappa X''(x) T(t)$$

分离变量（当 $X T \neq 0$）：
$$\frac{T'(t)}{\kappa T(t)} = \frac{X''(x)}{X(x)} = -\lambda$$

左边仅依赖 $t$，右边仅依赖 $x$，两者相等必然为同一常数，记作 $-\lambda$。

由此得到两个 ODE：
$$X''(x) + \lambda X(x) = 0 \quad \cdots (1)$$
$$T'(t) + \kappa \lambda T(t) = 0 \quad \cdots (2)$$

### 第二步：求解特征值问题

将分离假设代入边界条件 $u(0,t) = u(L,t) = 0$：
$$X(0)T(t) = 0, \quad X(L)T(t) = 0$$

由于 $T(t) \not\equiv 0$（否则解平凡），必须有：
$$X(0) = 0, \quad X(L) = 0$$

于是得到 **Sturm-Liouville 特征值问题**：
$$\begin{cases} X''(x) + \lambda X(x) = 0, \quad 0 < x < L \\ X(0) = X(L) = 0 \end{cases}$$

**分类讨论** $\lambda$ 的值：

- **$\lambda < 0$**：通解 $X(x) = A e^{\sqrt{-\lambda}x} + B e^{-\sqrt{-\lambda}x}$。代入 $X(0)=X(L)=0$ 得 $A = B = 0$（平凡解）。

- **$\lambda = 0$**：通解 $X(x) = A x + B$。$X(0)=0 \Rightarrow B=0$，$X(L)=0 \Rightarrow A=0$（平凡解）。

- **$\lambda > 0$**：通解 $X(x) = A \cos(\sqrt{\lambda}x) + B \sin(\sqrt{\lambda}x)$。
  $X(0)=0 \Rightarrow A=0$。
  $X(L)=0 \Rightarrow B \sin(\sqrt{\lambda}L)=0$。
  非平凡解要求 $\sin(\sqrt{\lambda}L)=0$，即 $\sqrt{\lambda}L = n\pi$（$n=1,2,3,\ldots$）。

因此：
$$\lambda_n = \left(\frac{n\pi}{L}\right)^2, \quad X_n(x) = \sin\frac{n\pi x}{L}, \quad n = 1, 2, 3, \ldots$$

### 第三步：求解时间部分

对每个 $\lambda_n$，代入 (2)：$T'(t) + \kappa\lambda_n T(t) = 0$，解得：
$$T_n(t) = C_n e^{-\kappa\lambda_n t} = C_n \exp\!\left(-\kappa\left(\frac{n\pi}{L}\right)^2 t\right)$$

### 第四步：叠加原理与初始条件匹配

由于方程和边界条件都是齐次的，任意有限线性组合仍是解。取无穷级数：
$$u(x,t) = \sum_{n=1}^\infty B_n \sin\frac{n\pi x}{L} \exp\!\left(-\kappa\left(\frac{n\pi}{L}\right)^2 t\right)$$

代入初始条件 $u(x,0) = f(x)$：
$$f(x) = \sum_{n=1}^\infty B_n \sin\frac{n\pi x}{L}$$

这正是 $f(x)$ 在 $[0, L]$ 上的**正弦 Fourier 级数**。由正交关系：
$$B_n = \frac{2}{L} \int_0^L f(x) \sin\frac{n\pi x}{L} \, dx, \quad n = 1, 2, 3, \ldots$$

---

## 解的物理解释

> [!info] 衰减速率
> 第 $n$ 个模式 $\sin(n\pi x/L)$ 的衰减时间尺度为 $e^{-\kappa (n\pi/L)^2 t}$。
> - **高模态（$n$ 大）**：空间振荡快，衰减极快 → 初始时刻的精细结构迅速平滑化
> - **低模态（$n$ 小）**：衰减慢 → 长时间行为由基模 $n=1$ 主导
> 
> 这解释了热方程的正则化效应：高频分量指数级衰减，解迅速变光滑。

> [!info] 热传导的渐近行为
> 当 $t \to \infty$：
> $$u(x,t) \sim B_1 \sin\frac{\pi x}{L} e^{-\kappa(\pi/L)^2 t}$$
> 温度趋于零（边界为零温度），衰减的速率由第一特征值 $\lambda_1 = (\pi/L)^2$ 决定。

## 例题

> [!example] 恒定初始温度的冷却
> 设初始温度均匀：$f(x) = T_0$（常数）。
> 
> $$B_n = \frac{2}{L} \int_0^L T_0 \sin\frac{n\pi x}{L} dx = \frac{2T_0}{n\pi}(1 - \cos(n\pi))$$
> 
> 当 $n$ 为偶数时 $\cos(n\pi) = 1$，故 $B_n = 0$；$n$ 为奇数时 $\cos(n\pi) = -1$，故 $B_n = \frac{4T_0}{n\pi}$。
> 
> $$u(x,t) = \frac{4T_0}{\pi} \sum_{k=0}^\infty \frac{1}{2k+1} \sin\frac{(2k+1)\pi x}{L} \exp\!\left(-\kappa\left(\frac{(2k+1)\pi}{L}\right)^2 t\right)$$
> 
> 仅奇数模态被激发，反映出均匀初值关于杆中点的对称性。

## 非齐次边界条件的处理

若边界条件非齐次：$u(0,t) = A$，$u(L,t) = B$（常数），可将解写为：
$$u(x,t) = v(x) + w(x,t)$$

其中 $v(x)$ 是**稳态解**（满足 $v''(x) = 0$，$v(0)=A, v(L)=B$）：
$$v(x) = A + \frac{B-A}{L}x$$

则 $w(x,t)$ 满足齐次边界条件的初边值问题：
$$\begin{cases}
w_t = \kappa w_{xx} \\[4pt]
w(0,t) = w(L,t) = 0 \\[4pt]
w(x,0) = f(x) - v(x)
\end{cases}$$

然后用上述分离变量法求解 $w$。

## 习题

1. **基本求解**：求解 $u_t = u_{xx}$，$0 < x < \pi$，$u(0,t) = u(\pi,t) = 0$，$u(x,0) = \sin(2x) - 3\sin(5x)$。

2. **分段初值**：长为 $L$ 的杆，初始温度 $f(x) = \begin{cases} x, & 0 \leq x < L/2 \\ L-x, & L/2 \leq x \leq L \end{cases}$（三角形分布），两端零温度。求 $u(x,t)$ 的级数形式。

3. **Neumann 边界**：求解 $u_t = \kappa u_{xx}$，$0 < x < L$，$u_x(0,t) = u_x(L,t) = 0$（绝热边界），$u(x,0) = f(x)$。注意特征函数变为 $\cos\frac{n\pi x}{L}$。

4. **非齐次边界**：求解 $u_t = u_{xx}$，$0 < x < 1$，$u(0,t) = 0$，$u(1,t) = 100$，$u(x,0) = 0$。求稳态解 $v(x)$ 和瞬态 $w(x,t)$。

5. **Robin 边界**：考虑 $u_t = u_{xx}$，$0 < x < 1$，$u(0,t) = 0$，$u_x(1,t) + h u(1,t) = 0$（$h>0$，对流冷却）。写出特征值方程并讨论 $\lambda$ 的定性行为。

6. **最大温度衰减**：利用最大值原理和分离变量法定性解释：为什么 $u(x,t) \to 0$ 当 $t \to \infty$ 时（零边界温度条件）？哪个特征模态衰减最慢？

7. **扩散时间尺度**：对长为 $L$ 的杆，定义"驰豫时间" $\tau = 1/(\kappa\lambda_1)$。解释其物理含义：为什么粗杆（$L$ 大）冷却更慢？为什么金属（$\kappa$ 大）冷却更快？

> [!hint]- 部分提示
> - 1: $u(x,t) = e^{-4t}\sin(2x) - 3e^{-25t}\sin(5x)$（初值已经是特征函数的线性组合）
> - 2: 正弦系数用分段积分计算
> - 3: 特征值为 $\lambda_n = (n\pi/L)^2$，$n = 0,1,2,\ldots$；$\lambda_0 = 0$ 对应常数模态，温度趋于平均值
> - 4: $v(x) = 100x$；$w(x,t)$ 用分离变量法，初值为 $-100x$
> - 5: 特征值方程为 $\sqrt{\lambda}\cos(\sqrt{\lambda}) + h\sin(\sqrt{\lambda}) = 0$
